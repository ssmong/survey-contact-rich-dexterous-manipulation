### LLARVA: Vision-Action Instruction Tuning Enhances Robot Learning

**전체 제목:** LLARVA: Vision-Action Instruction Tuning Enhances Robot Learning

**저자:** Dantong Niu, Yuvan Sharma, Giscard Biamby, Jerome Quenum, Yutong Bai, Baifeng Shi, Trevor Darrell, Roei Herzig (Berkeley AI Research, UC Berkeley)

**학회/연도:** arXiv 2024

**아키텍처:** Built on LLaVA 1.5 with a Llama 2 7B language model and CLIP ViT-L/14 vision encoder. Both vision and language encoders are frozen; adaptation is done via LoRA adapters on the LLM and a trainable MLP projection layer that aligns visual features to language token dimensionality. Actions are decoded as language tokens (no specialized action head) --- the model outputs both 2D visual trace predictions and action sequences as structured text. The total parameter count is ~7B (dominated by the Llama 2 backbone).

**행동 공간:** Varies by robot embodiment (joint or end-effector control, absolute or delta). The model handles heterogeneous action spaces across 13 robot types by encoding control mode information in the instruction prompt. Continuous actions are discretized into text tokens. Evaluation includes 7D EEF control (Franka) and joint-space control (RLBench).

**다지 핸드 지원:** ✗ --- Gripper-only evaluation across all 13 robot embodiments.

**힘/임피던스 출력:** ✗ --- Position targets only.

**핵심 방법론:** LLARVA introduces vision-action instruction tuning, where the VLA is trained to predict auxiliary 2D visual traces (sequences of (x,y) end-effector trajectory coordinates) alongside robot actions. These traces are generated by training a Faster R-CNN detector on end-effector bounding boxes across diverse robots, then extracting center-point trajectories. The instruction prompt encodes robot type, control mode, task description, and proprioceptive history, enabling a single model to handle 13 different robot embodiments. Training is two-stage: (1) vision-action instruction pretraining on 8.5M image-trace pairs from 37 OXE subsets, then (2) fine-tuning on downstream tasks.

**훈련 데이터:** Pretraining: 8.5M image-2D visual trace pairs from 37 Open X-Embodiment subsets spanning 13 robot types. Fine-tuning: RLBench (800 demonstrations per task, 12-18 tasks) and real Franka Panda (1,920 episodes for pick/stack/destack tasks).

**주요 기여:**
- Proposed vision-action instruction tuning with auxiliary 2D visual trace prediction as a self-supervised pretraining objective, improving action prediction by providing an intermediate spatial representation that bridges vision and action.
- Demonstrated that structured natural-language prompts encoding robot type, control mode, and proprioceptive history enable a single VLA to generalize across 13 robot embodiments without embodiment-specific heads.
- Showed that instruction pretraining on 8.5M OXE trace pairs provides +17.5% average improvement and visual trace prediction adds +15% on downstream tasks.

**한계점:**
- 2D visual traces lack depth information; authors note that 3D trace representations would improve fine-grained manipulation.
- Single-view 128x128 input images limit spatial precision for delicate tasks.
- Current architecture cannot process multiple views or leverage depth/voxel representations.
- RLBench performance (43.3%) lags behind 3D-based methods like PerAct (55.3%), highlighting the cost of being 2D-only.
- Limited temporal context for long-horizon tasks despite trace-based memory augmentation.
- No dexterous hand or force-aware evaluation.

## 데이터셋 / 데이터 수집

- **사용 데이터셋:** Open X-Embodiment (37 subsets, 13 robot types) for pretraining. RLBench simulation and real Franka Panda for fine-tuning.
- **수집 방법:** OXE data aggregated from diverse sources. Visual traces generated via Faster R-CNN end-effector detection on OXE episodes. RLBench demonstrations from simulation. Real-robot demonstrations via teleoperation.
- **데이터 규모:** 8.5M image-trace pairs for pretraining. 800 demonstrations per RLBench task. 1,920 episodes for real Franka tasks.
- **원격 조작 장비:** Varies by OXE constituent dataset. Real Franka: not specified.
- **데이터 포맷:** RLDS (OXE standard) for pretraining.
- **공개 여부:** OXE data is public. Model weights and code release status unclear.
