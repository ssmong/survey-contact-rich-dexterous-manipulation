# 9.3 MuJoCo Manipulus

- **全称：** MuJoCo Manipulus: A Dexterous Tool Manipulation Benchmark
- **作者：** Google DeepMind及合作者
- **发表venue/年份：** 2025（预印本）
- **仿真平台：** MuJoCo
- **灵巧手支持：** 多指手执行工具操作
- **任务：** 16个工具使用任务，包括锤击、拧螺丝、搅拌及其他功能性工具操作场景

## 核心方法/设计

Manipulus提供了一个专门针对灵巧工具操作的基准，这一领域在现有仿真基准中代表性不足。环境要求协调的多指控制来抓取、重定向和功能性地使用工具——超越了简单的拾放操作，涉及接触丰富的功能性交互。任务旨在同时测试工具操作的抓取阶段和功能使用阶段。

## 主要贡献

- 首个在MuJoCo中包含16个不同任务的灵巧工具操作专用基准
- 系统评估工具使用能力，要求兼具抓取稳定性和功能灵巧性
- 兼容标准MuJoCo生态系统（Gymnasium、dm_control）

## 局限性/不足

- 工具几何形状和物理属性相比真实工具有所简化
- 无可变形工具或工件交互（如切割、涂抹）
- 工具使用策略的仿真到真实验证有限

## 覆盖缺口

| 标准 | 是否覆盖？ |
|------|-----------|
| 力/力矩评估指标 | 否 |
| 可变形物体任务 | 否（仅刚性工具和工件） |
| 触觉传感 | 否 |
| 多阶段/长时程任务 | 部分（工具使用涉及抓取+功能使用，但任务并非显式的多阶段序列） |
| 多手协调 | 否 |

## 开源状态

开源。通过MuJoCo生态系统获取。
