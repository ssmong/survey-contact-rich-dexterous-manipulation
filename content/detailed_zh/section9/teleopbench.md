# 9.7 TeleOpBench

- **全称：** TeleOpBench: A Simulator-Centric Benchmark for Dual-Arm Dexterous Teleoperation
- **作者：** Hangyu Li, Qin Zhao, Haoran Xu 及合作者
- **发表venue/年份：** 2025（arXiv 2505.12748）
- **仿真平台：** Isaac Sim
- **灵巧手支持：** 3种具有双臂灵巧手的人形体
- **遥操作接口：** 基于相机（SMPLerX + MediaPipe）和基于VisionPro（Apple Vision Pro）
- **任务：** 30个双臂灵巧操作任务
- **评估：** 基准测试了4种遥操作模式

## 核心方法/设计

TeleOpBench提供了一个标准化基准，用于评估双臂灵巧操作的遥操作系统。它实现了两条动作捕捉流程：基于相机的流程使用SMPLerX和MediaPipe进行网络摄像头人体动作捕捉，以及基于VR的流程使用Apple Vision Pro。基准包括实时逆运动学处理、基于MeshCat的3D可视化和运动数据传输。通过标准化评估环境和指标，可以直接比较不同的遥操作方法。

## 主要贡献

- 首个具有30个标准化任务的双臂灵巧遥操作系统性基准
- 支持多种遥操作模式（基于相机和基于VR），可直接比较
- 实时逆运动学流程，将人手运动映射到多种机器人体
- 标准化的遥操作质量评估指标

## 局限性/不足

- 基准仅限仿真；评估流程不包含真实机器人执行
- 仅限两种遥操作输入模式；未覆盖基于手套和外骨骼的接口
- 未包含力反馈评估，尽管其与接触丰富操作高度相关
- 3种人形体可能无法覆盖真实世界灵巧平台的多样性

## 覆盖缺口

| 标准 | 是否覆盖？ |
|------|-----------|
| 力/力矩评估指标 | 否（遥操作质量指标不包含力调节评估） |
| 可变形物体任务 | 否 |
| 触觉传感 | 否 |
| 多阶段/长时程任务 | 部分（部分任务涉及顺序步骤，但无显式的长时程评估协议） |
| 多手协调 | 是（双臂灵巧操作是核心关注点） |
