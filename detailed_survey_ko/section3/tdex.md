## 3.20 T-Dex

- **전체 제목:** Dexterity from Touch: Self-Supervised Pre-Training of Tactile Representations with Robotic Play
- **저자:** Irmak Guzey, Ben Evans, Soumith Chintala, Lerrel Pinto (NYU, Meta)
- **학회/연도:** ICRA 2024 (arXiv 2303.12076, 2023년 3월)
- **arXiv:** https://arxiv.org/abs/2303.12076

**힘/촉각 입력 유형:** Allegro Hand 손끝에 장착된 DIGIT 비전 기반 촉각 센서. 비구조화된 놀이 데이터(2.5시간 자유 상호작용)에서의 자기 지도 사전 훈련을 통해 촉각 이미지 인코딩. 시간 대비 학습 목표로 훈련된 촉각 인코더가 동결 후 하류 조작 정책 입력으로 사용.

**힘/임피던스 출력:** 없음. 위치 전용 출력 (Allegro Hand 관절 위치 목표).

**로봇 플랫폼:** Allegro Hand (16 DoF) + DIGIT 촉각 센서 + Kinova Jaco 팔. Section 3에서 병렬 조 그리퍼가 아닌 정교한 다지 핸드를 사용하는 극소수 논문 중 하나.

> **주목:** Section 3에서 정교한 핸드(Allegro)에서 평가된 유일한 힘/촉각 인식 조작 논문 중 하나.

> **참고:** VLA 아님 (언어 조건부 없음). 힘/촉각 인식 정책과 정교한 조작 사이의 단절이라는 섹션의 핵심 질문에 직접 관련되어 포함.

**작업:** 5가지 정교한 조작 작업:
1. 조이스틱 밀기
2. 책 열기
3. 그릇 분리
4. 페그 삽입 (컵-홀더)
5. 점토 밀기

**핵심 방법론:** 2단계 접근:
1. **자기 지도 촉각 사전 훈련:** 2.5시간 비구조화 "놀이" 데이터(물체와의 무작위 상호작용) 수집. DIGIT 이미지에서 시간 대비 학습(TCN)으로 촉각 인코더 훈련. 작업 레이블 없이 컴팩트한 촉각 표현 생성.
2. **비파라메트릭 정책 학습:** 소수 작업 시연(5-10개)이 주어지면, 사전 훈련된 촉각 표현을 시각 관측(손목 카메라)과 결합. 공동 촉각-시각 임베딩 공간에서 최근접 이웃 검색을 통해 정책 학습.

**주요 기여:**
- 비구조화 놀이 데이터에서의 자기 지도 촉각 사전 훈련이 하류 정교한 작업에 유용한 표현을 생성함을 시연
- 촉각 기반 정책이 5가지 정교한 작업에서 비전 전용 및 관절 토크 기반 기준선을 평균 1.7배 능가
- 정교한 핸드(Allegro + DIGIT)에서의 최초 엔드-투-엔드 촉각 조작 시스템 중 하나

**한계점:**
- 촉각 입력에도 위치 전용 출력; 힘 조절 없음
- 비파라메트릭(최근접 이웃) 정책이 시연 분포 이상 일반화 제한
- 촉각 사전 훈련에 설정당 2.5시간 놀이 데이터 수집 필요
- 5개 작업 제한; 고힘 접촉이 풍부한 작업(도구 사용, 조립) 미평가
- 언어 조건부나 VLM 통합 없음

**정량적 결과:**

| 작업 | T-Dex (촉각+비전) | 비전 전용 | 토크 전용 |
|---|---|---|---|
| 조이스틱 | ~80% | ~45% | ~35% |
| 책 열기 | ~75% | ~50% | ~40% |
| 그릇 분리 | ~85% | ~55% | ~30% |
| 페그 삽입 | ~70% | ~40% | ~25% |
| 점토 | ~65% | ~35% | ~20% |

성공률은 근사값. T-Dex가 5개 작업 전체에서 기준선 대비 평균 1.7배 향상. 세밀한 접촉 센싱이 필요한 작업(페그 삽입, 점토)에서 향상 가장 큼.

## 추론 / 배포
- **추론 지연 시간:** 실시간 가능. 비파라메트릭 최근접 이웃 검색은 추론 시 빠름.
- **배포 하드웨어:** Allegro Hand + DIGIT 센서 + Kinova Jaco 팔. 시각 관측용 손목 장착 카메라.
- **실시간 가능 여부:** 예. 물리 하드웨어에서 실시간 배포.

## 데이터셋 / 데이터 수집
- **사용 데이터셋:** 맞춤 놀이 데이터셋(2.5시간 비구조화 상호작용) + 소수 작업 시연(작업당 5-10개).
- **수집 방법:** 작업 시연은 원격 조작; 놀이 데이터는 자율 무작위 상호작용.
- **데이터 규모:** 2.5시간 놀이 데이터 + 5개 작업에 걸쳐 ~50개 작업 시연.
- **공개 여부:** 코드 공개: [GitHub](https://github.com/irmakguzey/tdex) (프로젝트 페이지: https://tactile-dexterity.github.io/).
