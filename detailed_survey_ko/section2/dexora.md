### 2.4 Dexora

**전체 제목:** Dexora: Open-Source VLA for High-DoF Bimanual Dexterity

**저자:** Dexora Team (Tsinghua University; 연락처: pjr24@mails.tsinghua.edu.cn)

**학회/연도:** ICRA 2026

**GitHub:** [ZZongzheng0918/Dexora](https://github.com/ZZongzheng0918/Dexora)

**핸드 하드웨어:** 이중 정교한 핸드를 갖춘 양팔 시스템, 총 36 DoF. 원격 조작은 팔 제어를 위한 외골격과 정교한 핸드 제어를 위한 Apple Vision Pro를 사용.

**작업 (범주별 데이터셋 비율):**
- 집기-놓기 (데이터셋의 55%)
- 정교한 조작 (20%)
- 조립 (15%)
- 관절 물체 조작 (10%)

**핵심 방법론:** Dexora는 고DoF 양손 정교한 조작을 위해 대규모 원격 조작 실세계 데이터와 시뮬레이션 훈련 데이터를 결합하는 오픈 소스 VLA 프레임워크이다. 하이브리드 데이터 수집 접근법은 팔 제어를 위한 외골격 원격 조작과 정교한 손가락 제어를 위한 Vision Pro 핸드 트래킹을 사용한다. 4개 카메라 시점에서의 다시점 동기화 녹화가 로봇 고유감각 피드백과 함께 시연을 캡처한다. 시스템은 데이터 포맷 준수를 위해 LIBERO-2.1 표준을 따른다.

**아키텍처/파라미터:**
- VLA 아키텍처 세부 사항은 서베이 시점에서 저장소에 완전히 공개되지 않음
- 훈련 데이터: 12.2K 실세계 원격 조작 에피소드 + 10만 시뮬레이션 에피소드
- 실세계 데이터: 약 40.5시간
- 물체 목록: 재현성을 위한 구매 링크가 포함된 347개 물체 세트
- 다시점 입력: 4개 동기화된 카메라 시점

**주요 기여:**
- 대규모 오픈 소스 양손 정교한 데이터셋: 12.2K 실세계 에피소드(40.5시간) + 10만 시뮬레이션 에피소드, 구매 링크(타오바오/아마존)가 포함된 347개 물체 목록
- 고품질 양손 정교한 시연을 위해 외골격(팔)과 Vision Pro(손)를 결합하는 하이브리드 원격 조작 파이프라인
- LIBERO-2.1 표준 준수와 HuggingFace에서의 전체 데이터 공개로, 재현 가능한 정교한 VLA 연구의 진입 장벽을 낮춤

**한계점:**
- arXiv 링크가 서베이 시점에서 플레이스홀더 형식("2026.xxxxx")이며, 전체 논문이 아직 공개되지 않았을 수 있음
- 현재 저장소에서 VLA 아키텍처 세부 사항이 부족
- 데이터 수집 파이프라인에 힘이나 촉각 센싱 없음; 시연은 관절 위치와 시각 데이터만 캡처
- 36-DoF 양팔 시스템이 특정적; 다른 핸드 플랫폼으로의 교차 구현체 전이는 다루지 않음

**결과:**

> **[편집 주 -- 작업 성공률 미보고]:** 이 항목은 데이터셋 통계(에피소드 수, 시간, 물체 수)만 포함하며 작업 성공률이나 조작 성능 지표가 없다. 작업 범주별(집기-놓기, 정교한 조작, 조립, 관절 물체 조작) 성공률 없이는 VLA의 실제 조작 능력을 평가하는 것이 불가능하다. 전체 논문이나 저장소에서 정량적 작업 성능을 확인해야 한다.

## 추론 / 배포

- **추론 지연 시간:** 보고되지 않음. 저장소와 이용 가능한 문서에서 추론 지연이나 제어 주파수 미공개.
- **배포 하드웨어:** 이중 정교한 핸드(총 36 DoF)를 갖춘 양팔 시스템. 데이터 수집은 외골격(팔) + Apple Vision Pro(손) 사용. 추론 하드웨어 미명시.
- **실시간 가능 여부:** 미검증. 공개 자료에 추론 지연이나 배포 하드웨어 정보 없음.

## 데이터셋 / 데이터 수집

- **사용 데이터셋:** Dexora 데이터셋 (본 연구와 함께 공개된 맞춤). 두 구성 요소: 실세계 원격 조작 에피소드와 시뮬레이션 에피소드.
- **수집 방법:** 외골격(팔 제어, 팔당 7 DoF)과 Apple Vision Pro(정교한 핸드 제어)를 결합한 하이브리드 원격 조작 시스템. 4개 카메라 시점에서의 다시점 동기화 녹화와 로봇 고유감각 피드백.
- **데이터 규모:** 12.2K 실세계 원격 조작 에피소드 (292만 프레임, ~40.5시간) + 10만 시뮬레이션 에피소드 (곧 공개 예정). 17개 의미 범주에 걸쳐 재현성을 위한 구매 링크(타오바오/아마존)가 포함된 347개 물체.
- **원격 조작 장비:** 팔 제어를 위한 외골격 + 정교한 핸드 트래킹을 위한 Apple Vision Pro (총 36 DoF 양팔 시스템).
- **데이터 포맷:** LIBERO-2.1 표준. 에피소드에는 다시점 RGB 관측, 로봇 상태(관절 위치/속도), 행동 명령, 궤적당 5개 자연어 작업 설명 포함. 에피소드 중심과 작업 수준 구성 뷰 모두 이용 가능.
- **공개 여부:** 예. 실세계 데이터셋(12.2K 에피소드)이 HuggingFace에 공개. 모델 가중치도 HuggingFace에. 코드: [GitHub](https://github.com/ZZongzheng0918/Dexora). 시뮬레이션 데이터셋 공개 예정.
