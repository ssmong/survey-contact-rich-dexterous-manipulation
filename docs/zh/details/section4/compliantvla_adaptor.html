4.1 CompliantVLA-adaptor

完整标题： Compliant Action Generation via VLA Adaptor for Impedance-Controlled Robot Manipulation

作者： Luigi Penco, Niko Suenderhauf, Fares Abu-Dakka (IIT Genoa / TU Darmstadt)

发表venue/年份： arXiv预印本，2026

K/D如何确定： VLM (GPT-4o-mini) 零样本提示。VLM接收当前场景图像和任务描述，然后输出每轴笛卡尔刚度和阻尼值。无需训练或微调VLM——该系统完全无需训练，完全依赖VLM对不同任务和接触条件下适当柔顺性水平的零样本推理。

输出内容： 6D笛卡尔阻抗参数：平移刚度（Kx, Ky, Kz）和平移阻尼（Dx, Dy, Dz）。这些参数被输入笛卡尔阻抗控制器，用于跟踪由冻结VLA（如OpenVLA或pi0）生成的位置轨迹。

机器人平台： Franka Emika Panda + 平行夹爪。无灵巧手。

任务： 接触丰富的操作任务，包括：擦拭表面、插头插入、抽屉开关、带柔顺性的物体交接。在LIBERO仿真基准测试和真实Franka硬件上均进行了评估。

核心方法： 该系统引入一个"适配器"模块，位于冻结VLA（生成位置轨迹）和机器人阻抗控制器之间。VLM从视觉输入推理任务语义和接触需求，并生成适当的刚度/阻尼配置，无需任何特定任务训练。来自VLA的位置轨迹和来自VLM适配器的阻抗参数在阻抗控制器层面结合，在保持VLA任务执行能力的同时实现柔顺行为。

架构/参数： 双模块流水线：(1) 冻结的预训练VLA（OpenVLA 7B或pi0 3.3B）生成末端执行器位置轨迹，(2) GPT-4o-mini作为阻抗适配器通过API调用生成K/D值。笛卡尔阻抗控制器运行在Franka的实时控制接口上。不涉及额外的神经网络训练。

主要贡献：

  • 提出用VLM生成的阻抗参数增强冻结VLA，在不重新训练VLA的情况下实现柔顺操作
  • 证明VLM可以仅从视觉和语言上下文零样本推理接触丰富任务的适当刚度/阻尼
  • 提供模块化、无需训练的方法，可以为任何位置轨迹VLA添加柔顺性能力

局限性/差距：

  • 不支持灵巧手——仅在Franka Panda上使用夹爪
  • K/D值按任务段生成，不在接触过程中连续自适应——本质上是开环阻抗调度
  • 依赖商业API（GPT-4o-mini），有相关延迟和成本
  • VLM生成的阻抗值无正式的稳定性或无源性保证
  • 评估中任务多样性有限；未涉及高力工业任务
  • VLM推理延迟（约100ms-1s）将阻抗更新限制在约1-10 Hz，比实时阻抗控制器所需的100-1000 Hz低三个数量级。

结果： 在LIBERO中，接触丰富任务的成功率相比仅位置VLA基线有所提升。柔顺适配器降低了插入和交接任务中的接触力。在Franka Panda上的实际机器人实验证实，VLM生成的阻抗参数可产生定性上适当的柔顺行为。

推理 / 部署

  • 推理延迟： 未明确进行基准测试。VLM适配器（GPT-4o-mini API调用）每次阻抗参数更新增加约100ms-1s。底层VLA（OpenVLA或pi0）以其原生速度进行轨迹生成。阻抗参数按任务段生成，而非按控制步生成。
  • 部署硬件： Franka Emika Panda及其实时控制接口。VLM推理通过云API（GPT-4o-mini）运行；适配器模块无需设备端GPU。VLA推理硬件未指定。
  • 是否支持实时？ 部分支持。来自VLA的位置轨迹和底层阻抗控制器以实时速率运行（Franka上1 kHz）。但VLM阻抗更新因API延迟限制在约1-10 Hz，使得快速接触过程中的连续阻抗自适应不可行。

数据集 / 数据采集

  • 使用的数据集： LIBERO基准测试（8个任务中的6个）和ManiSkill基准测试（8个任务中的2个）用于仿真评估。发布了用于RAG组件的自定义力-力矩-阻抗-场景上下文数据集。
  • 采集方法： LIBERO/ManiSkill评估使用仿真rollout；Franka Panda上的真实世界演示用于微调（每个任务50次成功试验）。视觉输入来自Logitech USB相机（全局视角）和Intel RealSense D435i（末端执行器安装），图像调整为224x224。
  • 数据规模： 每个任务50次演示用于真实世界微调；在8个代表性任务上进行仿真评估。
  • 遥操作设备： 不适用（VLM系统无需训练；用于VLA微调的演示采集方法未指定）。
  • 数据格式： RGB图像 + 本体感觉状态 + 语言指令；力-力矩-阻抗上下文数据集格式未指定。
  • 是否公开可用？ 代码、提示词和力-力矩-阻抗-场景上下文数据集发布于 https://sites.google.com/view/compliantvla
