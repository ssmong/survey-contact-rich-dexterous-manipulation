# 11.1 LEAP Hand V2

  • 全称： LEAP Hand V2: Advanced Dexterous Robotic Hand
  • 作者： Kenneth Shaw, Deepak Pathak (CMU)
  • 发表场所/年份： 2024（V1版：RSS 2023）
  • 自由度： 16个驱动自由度 + 可活动掌部
  • 执行器： Dynamixel舵机
  • 成本： ~$3,000
  • 重量： 轻量化（可安装于标准机械臂）
  • 触觉传感： 无内置（可加装第三方传感器）
  • 仿真模型： MuJoCo, Isaac Lab

定量参数

参数 数值
执行器控制带宽 ~50 Hz有效控制频率（受Dynamixel舵机限制）
最大指尖力 现有文献未报告
耐久性 多个研究组已验证数千次抓取循环（CrossDex、SeqDex、HandelBot、DexCap、ComFree-Sim）

核心方法/设计

LEAP Hand V2在初代LEAP Hand（RSS 2023）基础上进行了改进，包括优化机械设计、更强的执行器以及可活动掌部以增强抓取稳定性。该手使用Dynamixel舵机驱动所有关节，提供位置和电流（力矩近似）反馈。设计强调研究可及性：所有结构件均可3D打印或使用现成组件，物料清单完整公开。V2版新增的可对置掌关节扩大了抓取包络，使操作模式更接近人手。

主要贡献

  • 研究社区中采用最广泛的低成本灵巧手之一（被CrossDex、SeqDex、HandelBot、DexCap、ComFree-Sim等使用）
  • 可活动掌部（V2版）扩展了V1固定掌部设计之外的抓取类型
  • 多个独立研究组已验证从仿真到真实的零样本迁移流程
  • 鲁棒的机械设计，在研究环境中经受住了数千次抓取循环

局限性/不足

  • 无集成触觉传感；指尖接触仅依赖位置控制
  • Dynamixel舵机带宽有限（~50 Hz有效控制频率），低于定制执行器
  • $3,000的成本虽然对灵巧手而言较低，但在本节中最高
  • 16自由度少于人手（~20-25个功能性自由度），对某些任务限制了操作灵巧性

开源状态

完全开源（CAD、固件、软件）。项目主页：v2-adv.leaphand.com
