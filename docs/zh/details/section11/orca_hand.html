# 11.2 ORCA Hand

  • 全称： ORCA Hand: An Affordable and Dexterous Robotic Hand for Contact-Rich Manipulation
  • 作者： ETH Zurich（机器人系统实验室）
  • 发表场所/年份： 2025
  • 自由度： 17个驱动自由度

  • 仿真到真实： 已验证，训练时间1小时

核心方法/设计

ORCA Hand是一款17自由度灵巧手，专为接触密集型操作研究而设计。其标志性特点是集成式触觉传感——从设计阶段就将触觉传感器内置于指尖，而非后期加装。手的机械设计强调鲁棒的接触交互，柔顺指尖结构提高了对不同物体的抓取稳定性。配套的仿真到真实迁移流程表明，强化学习策略可在约1小时的总训练时间内完成训练和迁移。

主要贡献

  • 从设计阶段即集成触觉传感——本综述中唯一低于$2K且内置触觉的灵巧手
  • 17自由度，灵巧性略高于16自由度的替代方案（LEAP Hand）
  • 快速仿真到真实迁移流程（1小时训练），展示了实际部署能力
  • 完全开源的硬件设计，支持社区复现

局限性/不足

  • 较新的设计，社区采纳度和验证程度低于LEAP Hand
  • 触觉传感器规格（分辨率、模态）在现有文献中未充分表征
  • <$2K的成本为估计值；实际构建成本取决于本地元器件采购情况

开源状态

完全开源（硬件 + 软件）。项目主页：orcahand.com
