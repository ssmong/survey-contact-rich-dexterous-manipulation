# 11.5 Digit 360

  • 全称： Digit 360: A Multimodal Tactile Fingertip Sensor
  • 作者： Meta FAIR
  • 发表场所/年份： 2024
  • 类型： 触觉传感器（指尖模块），非完整手
  • 感知模态： 18+种模态，包括法向/剪切力、振动、温度、接近感知和高分辨率触觉图像
  • 成本： 未公开
  • 目标集成： 任何灵巧手平台（设计为模块化指尖）

定量参数

参数 数值
感知模态 18+种同步模态（法向力、剪切力、振动、温度、接近感知、触觉图像等）
外形尺寸 指尖大小，设计用于替换现有灵巧手的指尖
计算开销 较大——18+种模态需要并行数据处理
可获取性 尚未公开发售（截至综述日期，计划开源发布）

核心方法/设计

Digit 360不是一款灵巧手，而是一款设计用于集成到灵巧手中的多模态触觉指尖传感器。它扩展了Meta的DIGIT系列视觉触觉传感器，将感知模态从单一的基于相机的触觉图像大幅扩展至18+种同步模态，包括空间分布的法向力和剪切力、高频振动、温度和接近感知。该传感器采用紧凑外形，设计用于替换现有灵巧手的标准指尖。

主要贡献

  • 目前报告的最全面的触觉指尖传感器之一，具备18+种同步感知模态
  • 在单个指尖尺寸的封装中集成多模态感知（力、振动、温度、接近）
  • 设计为模块化升级，可集成到现有灵巧手平台
  • 解决了当前低成本灵巧手（LEAP、RUKA、ISyHand均缺乏触觉）的感知缺口

局限性/不足

  • 非完整手；需要集成到现有灵巧手平台
  • 成本未公开；可能远高于手本身的价格
  • 尚未广泛面向研究社区发售（计划开源发布）
  • 同时处理18+种模态的数据引入了计算开销
  • 尚未验证与现有触觉手的仿真到真实迁移流程的集成

开源状态

计划开源发布（截至综述日期，尚未公开发售）。
