# 11.8 XHand / Inspire Hand

  • 全称： XHand（亦称X-Hand）和Inspire Hand（亦称Inspire-Robots灵巧手）
  • 制造商： 中国机器人制造商（Inspire-Robots生产Inspire Hand；多家制造商生产XHand变体）
  • 年份： 约2023年起上市；2024-2025年在研究中的采纳率快速增长
  • 自由度： Inspire Hand：~12个驱动自由度（5指，独立屈伸 + 部分内收/外展）；XHand：~12-16自由度（因版本而异）
  • 成本： 显著低于西方制造的替代品（估计$1,000-3,000范围；具体价格因供应商和配置而异）
  • 触觉传感： 无标配（部分版本支持可选触觉模块）
  • 驱动方式： 直流电机配齿轮减速；位置控制

定量参数

参数 数值
手指 5个（仿人布局）
驱动自由度（Inspire） ~12（独立手指屈伸 + 拇指对置）
驱动自由度（XHand） ~12-16（因版本而异）
重量 ~500-800g（因版本而异）
抓握力 中等（满足典型操作任务需求）
控制接口 串口/CAN总线；部分版本提供ROS驱动
仿真模型 MuJoCo, IsaacGym（社区贡献的URDF模型）

核心方法/设计

XHand和Inspire Hand是中国制造的灵巧手，凭借合理的成本、足够的自由度和商业可获取性在机器人研究社区获得了广泛采纳。与开源研究手（LEAP、ORCA、ISyHand、RUKA、FAIVE）不同，这些是具有专业级制造水准的商业量产产品。它们填补了一个实际需求空间：比Shadow Hand（~$100K+）或Allegro Hand（~$15K+）更经济，无需组装即可商业购买，且具备足够的自由度（12-16）进行有意义的灵巧操作研究。在文献中Inspire Hand和XHand有时被互换使用，但它们来自不同制造商且有不同的规格。

在本综述中的使用情况

这些手频繁出现在本综述各处，尽管此前没有专门的硬件条目：

  • DexMachina（第1节）：Inspire Hand和XHand均用作双臂操作的目标本体形态
  • ManipTrans（第1节）：Inspire Hand和XHand在6款手中用于跨本体形态运动迁移
  • DexUMI（第1节）：XHand和Inspire Hand用于真实世界灵巧操作（CoRL 2025最佳论文入围）
  • UniDex-VLA（第2节）：Inspire和XHand均纳入8手FAAS框架
  • iDP3（第6节）：Inspire Hand（25自由度配置）搭载于Fourier GR1人形机器人
  • Dex1B（第8节）：Inspire Hand纳入1B规模演示数据集

主要贡献（作为研究平台）

  • 近期研究中采纳最广泛的商用灵巧手之一，尤其在中国机器人研究社区
  • Shadow/Allegro之外的高性价比替代方案，适合需要多指操作硬件的实验室
  • 已验证的跨本体形态兼容性：成功集成到重定向框架（ManipTrans、UniDex-VLA FAAS）中，与Shadow、Allegro和LEAP并用
  • 商业可获取性消除了组装时间和校准工作
  • 社区贡献的URDF/MJCF仿真模型生态持续增长

局限性/不足

  • 论文间的规格报告不一致：自由度计数不同（"Inspire Hand"报告为12到25不等，取决于被动关节的计数方式），使直接比较困难
  • 无标准化仿真模型：URDF/MJCF文件为社区贡献，可能与实际硬件不完全匹配
  • 闭源硬件：无CAD文件或固件修改的可能性
  • 标准配置无集成触觉传感
  • 文档和技术支持主要为中文；英文文档可能有限
  • 多家制造商和版本造成混淆："XHand"和"Inspire Hand"有时被混为一谈，尽管是不同产品
  • 不同生产批次和供应商之间的制造质量和一致性可能有差异
  • 执行器参数（带宽、力、重复性）缺乏独立基准测试

开源状态

闭源商业产品。Inspire Hand可从 inspire-robots.com 购买。XHand可从多家中国机器人供应商购买。部分研究仓库（ManipTrans、DexMachina）提供社区贡献的URDF模型。
