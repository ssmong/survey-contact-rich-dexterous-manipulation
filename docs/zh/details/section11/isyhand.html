# 11.3 ISyHand

  • 全称： ISyHand: An Intrinsically Soft Dexterous Hand for Contact-Rich Manipulation
  • 作者： MPI for Intelligent Systems (MPI-IS), Tubingen
  • 发表场所/年份： 2025
  • 自由度： 18（12个手指自由度 + 6个掌部/腕部自由度）
  • 成本： ~$1,300
  • 触觉传感： 无内置
  • 驱动方式： 腱驱动，内在柔顺关节

定量参数

参数 数值
执行器控制带宽 现有文献未报告
最大指尖力 现有文献未报告（柔顺关节可能限制最大力输出，低于刚性关节设计）
耐久性 现有文献未报告（腱磨损是潜在维护问题）

核心方法/设计

ISyHand采用了与LEAP等刚性关节手截然不同的机械方案：使用内在柔顺关节和腱驱动执行机构。柔顺性内嵌于关节机构本身，而非通过控制实现。这意味着手在抓取过程中会被动地适应物体几何形状——一种无需显式力控制的物理阻抗形式。18自由度设计（12个手指 + 6个掌部/腕部）提供了较高的灵巧性，腱驱动执行方式允许将电机布置在前臂，降低了指尖重量和惯量。

主要贡献

  • 内在机械柔顺性，无需显式阻抗控制即可实现被动力调节
  • 在本综述低于$2K的灵巧手中自由度最高（18）
  • 以最低成本（$1,300）实现最高自由度，性价比优异
  • 腱驱动执行方式实现低惯量的轻质指尖

局限性/不足

  • 无触觉传感；柔顺关节提供隐式接触适应但无显式接触测量
  • 腱驱动执行引入了线缆布线复杂性和潜在维护问题（腱磨损、松弛）
  • 由于难以准确建模腱动力学和关节柔顺性，软体/柔顺手的仿真到真实迁移更具挑战性
  • 柔顺关节可能限制最大指尖力，低于刚性关节设计
