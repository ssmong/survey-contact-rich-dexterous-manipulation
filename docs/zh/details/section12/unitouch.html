# 12.2 UniTouch

  • 全称： UniTouch: Universal Touch Representation for Robot Manipulation
  • 作者： Che Fang及合作者 (UCSD)
  • 发表场所/年份： CVPR 2024
  • 支持的传感器： 多种视觉触觉传感器
  • 对齐方式： 触觉-视觉-语言-声音多模态对齐
  • 预训练方法： 对比学习，将触觉表征与视觉、语言和音频模态对齐

核心方法/设计

UniTouch通过将触觉信号与其他三种模态——视觉、语言和声音——对齐来学习统一的触觉表征。使用对比学习（CLIP风格），UniTouch将触觉图像映射到一个与视觉图像、文本描述和接触事件录音共享的嵌入空间中。这种多模态对齐实现了零样本迁移：触觉表征可以用语言查询（"粗糙表面"）或与视觉外观匹配，而无需触觉特定标注。该方法源于触觉、视觉、语言和声音提供物理接触互补描述的洞察。

主要贡献

  • 首个触觉-视觉-语言-声音对齐模型，实现跨模态查询（如从视觉或语言查询"找到摸起来像砂纸的纹理"）
  • 通过语言-触觉对齐实现零样本触觉分类，无需任务特定训练
  • 统一的表征空间使触觉数据能够受益于语言和视觉编码器的丰富语义
  • 在多个下游任务上进行了验证，包括材料分类和接触属性估计

局限性/不足

  • 多模态对齐质量严重依赖配对的触觉-视觉-语言-声音数据的可用性和质量，而此类数据稀缺
  • 声音模态对冲击/接触事件有用，但对准静态操作（抓取、滑动）信息量较少
  • 表征针对分类/检索任务优化；在闭环操作策略学习中的效用未得到验证
  • 仅限于视觉触觉传感器；不支持非视觉触觉模态

共同局限性

与本节所有触觉表征模型一样，UniTouch未展示集成到闭环操作策略中的能力。详见Sparsh中对此共同局限性的详细讨论。
