2.3 DexVLA

全称： DexVLA: Vision-Language Model with Plug-In Diffusion Expert for Dexterous Manipulation

作者： Junjie Wen, Yichen Zhu, Jinming Li, Zhibin Tang, Chaomin Shen, Feifei Feng

发表venue/年份： CoRL 2025

arXiv： 2502.05855

手部硬件： Inspired Dexterous Hand（6自由度手部关节），安装在 Franka Emika 机械臂上。动作空间为12维：SE(3) 末端执行器位姿（3D位置 + 3D姿态）加6D手部关节空间。跨本体评估还涵盖单臂夹爪和双臂 UR5e 设置。

任务：

  • 通过课程学习的灵巧技能习得
  • 多指操作任务
  • 灵巧手抓取-放置
  • 手内操作

核心方法： DexVLA 引入了一种插件式架构，将10亿参数的扩散专家附加到冻结的视觉-语言模型上。VLM 骨干提供视觉理解和语言锚定，同时在灵巧训练期间保持冻结。扩散专家基于 VLM 的潜在表征生成高维灵巧手动作。这种设计避免了 VLM 预训练知识的灾难性遗忘，同时实现精确的灵巧控制。课程学习策略逐步增加任务难度。

架构/参数：

  • VLM 骨干：冻结的预训练视觉-语言模型（骨干规模在摘要层面未公开指定）
  • 动作头：ScaleDP（可扩展扩散策略），提供两种变体：
  • ScaleDP-H（大型变体）
  • ScaleDP-L（轻量变体）
  • 扩散专家约10亿参数
  • ScaleDP-H 和 ScaleDP-L 检查点均在 HuggingFace 上提供

主要贡献：

  • 插件式扩散专家架构：10亿参数扩散模型附加到冻结 VLM，在不损害 VLM 预训练能力的情况下实现灵巧操作（此声称在本条目中未经独立验证）
  • ScaleDP（可扩展扩散策略）：为高维灵巧动作空间设计的可扩展扩散动作生成模块，具有大型（H）和轻量（L）两种变体
  • 基于课程的训练策略，逐步增加灵巧任务复杂度，提升样本效率和最终性能

局限性/不足：

  • 仅输出位置目标；无力、阻抗或柔顺性输出
  • 10亿参数扩散专家在冻结 VLM 之上增加了显著的计算开销
  • 跨不同灵巧手型的评估并非主要关注点（不同于 UniDex-VLA 的8种手评估）
  • 无触觉或力/力矩传感器集成
  • 与仿真相比，真实世界评估范围较窄

训练效率： 仅训练扩散专家的速度为0.89轮/小时，而训练整个 VLA 为0.32轮/小时（2.78倍加速）。

推理 / 部署

  • 推理延迟： 在单块 NVIDIA A6000 GPU 上以 60 Hz 运行。这是所调研文献中报告的最快 VLA 推理速率之一。
  • 部署硬件： 单块 NVIDIA A6000 GPU 用于推理。尽管扩散专家扩展到10亿参数，系统仍实现了高效训练和快速推理。
  • 是否支持实时？ 是。在单块 A6000 GPU 上60 Hz 足以满足实时灵巧操作控制，超过大多数基于 VLA 系统典型的10--30 Hz 控制频率。

数据集 / 数据采集

  • 使用的数据集： 跨本体训练数据，涵盖单臂、双臂和灵巧手机器人。无单一命名基准；使用多种数据来源的组合。
  • 采集方法： 跨本体数据采集。具体采集方法（遥操作设备、仿真等）在公开材料中未详述。项目页面提及跨本体学习。
