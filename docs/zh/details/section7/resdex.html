7.1 ResDex

完整标题： ResDex: Residual Policy Learning for Dexterous Grasping with Large-Scale Object Training

作者： Haoyu Xiong, Yufei Wang, Jiayi Chen, Yi Wu

发表venue/年份： ICLR 2025

RL算法： PPO + 混合专家（MoE）架构和DAgger蒸馏；在基础策略之上使用残差策略来处理多样物体

手部硬件： Shadow Hand（24自由度）

仿真平台： IsaacGym Preview 4

仿真到真实？ 否

物体数量： 3200个物体；88.8%总体成功率（仿真）

任务： 在前所未有的规模上进行多样物体的灵巧抓取（3200个物体）

核心方法论： 通过将策略分解为基础抓取策略和使用混合专家架构的残差校正模块，将灵巧抓取扩展到数千个物体。MoE支持跨物体类别的专业化，同时共享通用抓取骨干。DAgger蒸馏将集成压缩为单一可部署策略。

主要贡献：

  • 将灵巧抓取训练扩展到3200个多样物体，为发表时最大的物体集
  • 用于处理物体多样性的残差策略 + MoE架构，避免灾难性遗忘
  • 在全部3200物体集上以单一策略达到88.8%成功率（仿真）

定量结果：

指标 数值
总体成功率 88.8%（仿真）
物体集大小 3200
手部 Shadow（24自由度）

局限性/差距： 仅仿真（未展示仿真到真实迁移）；使用单一手部构型（Shadow）；在仿真中依赖真值物体状态；未评估在扰动下的抓取稳定性

仅仿真限制： 仅在仿真中评估；未展示仿真到真实迁移，真实世界可行性未验证。

推理 / 部署

  • 推理延迟： 未报告。蒸馏后策略（MLP）每次前向传播 <1ms。
  • 部署硬件： 仅仿真（IsaacGym）。未展示真实机器人部署。
  • 支持实时？ 是，就策略本身而言（MLP推理速度极快）。但仅进行了仿真评估。

数据集 / 数据收集

  • 使用的数据集： 无预收集数据集。纯RL（PPO）+ MoE架构——所有数据在仿真中生成。
  • 收集方法： IsaacGym中的纯RL。残差策略 + 混合专家在3200个物体上训练。DAgger蒸馏将MoE集成压缩为单一策略。在仿真中使用真值物体状态。
  • 数据规模： 3200个物体用于训练。标准并行RL训练规模。
  • 遥操作设备： 不适用（纯RL，无演示）。
  • 数据格式： 不适用（在线RL，无离线数据集）。
  • 是否公开？ 数据集/策略发布状态未报告。
