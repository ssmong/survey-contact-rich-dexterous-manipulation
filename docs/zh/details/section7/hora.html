7.2 Hora

完整标题： In-Hand Object Rotation via Rapid Motor Adaptation

作者： Haozhi Qi, Ashish Kumar, Roberto Calandra, Yi Ma, Jitendra Malik

发表venue/年份： CoRL 2022

RL算法： PPO + 快速运动自适应（RMA）；训练基础策略和在线自适应模块，从近期交互历史中推断环境参数

手部硬件： Allegro Hand（16自由度）

仿真平台： IsaacGym Preview 4

仿真到真实？ 是；使用RMA进行在线自适应的零样本仿真到真实迁移（真实）。训练期间对物理参数（质量、摩擦、尺度）进行域随机化；自适应模块在部署时从本体感觉历史推断这些参数

物体数量： 多种物体包括立方体、圆柱体和YCB物体；在真实世界中评估了约10+物体

任务： 围绕指定轴的连续手内物体旋转（手掌中物体的z轴旋转）

核心方法论： 基于快速运动自适应（RMA）进行手内操作。在随机化环境中使用PPO训练基础策略。自适应模块（小型MLP）被训练为从短期本体感觉观察历史中预测潜在环境参数。在部署时，自适应模块提供隐式系统辨识，使策略能够适应真实世界动力学而无需显式校准。

主要贡献：

  • 首次在真实Allegro Hand上展示用于灵巧手内旋转的快速运动自适应（真实）
  • 在线自适应消除了手动系统辨识或手动调节域随机化的需求
  • 在真实硬件上对多样物体形状和尺寸实现鲁棒零样本迁移（真实）

局限性/差距： 仅限单轴旋转（非任意SE(3)重定向）；仅依赖本体感觉（无视觉或触觉）；物体必须保持掌心朝上的姿态；仅限刚性物体

推理 / 部署

  • 推理延迟： 未明确报告。带RMA自适应模块的MLP策略每次前向传播 <1ms，支持高频控制。
  • 部署硬件： Allegro Hand（16自由度）。策略在IsaacGym中训练；通过零样本仿真到真实迁移加在线RMA自适应部署。MLP策略足够轻量，可在任何计算平台上运行。
  • 支持实时？ 是。基于MLP的RL策略加轻量RMA模块可在Allegro Hand上实时运行。

数据集 / 数据收集

  • 使用的数据集： 无预收集数据集。纯RL（PPO）+ 快速运动自适应（RMA）——所有数据在仿真中生成。
  • 收集方法： IsaacGym中的纯RL，对物理参数（质量、摩擦、尺度）进行域随机化。在线自适应模块从本体感觉历史训练。零样本仿真到真实迁移。立方体、圆柱体和YCB物体。
  • 数据规模： IsaacGym中标准并行RL训练。真实世界评估约10+物体。
  • 遥操作设备： 不适用（纯RL，无演示）。
  • 数据格式： 不适用（在线RL，无离线数据集）。
  • 是否公开？ 代码发布状态未报告。
