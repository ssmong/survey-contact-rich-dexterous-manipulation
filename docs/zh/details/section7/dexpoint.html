7.1 DexPoint

完整标题： DexPoint: Generalizable Point Cloud Reinforcement Learning for Sim-to-Real Dexterous Manipulation

作者： Yuzhe Qin, Binghao Huang, Zhao-Heng Yin, Hao Su, Xiaolong Wang

发表venue/年份： CoRL 2022

arXiv： https://arxiv.org/abs/2211.09423

RL算法： PPO + 点云观测。使用想象手部点云作为增强输入，基于接触的奖励整形用于高效学习。

手部硬件： Allegro Hand（16自由度）用于真实世界实验

仿真平台： IsaacGym（NVIDIA GPU加速仿真）

仿真到真实？ 是。Allegro Hand上零样本仿真到真实迁移。关键仿真到真实创新：(1) 点云表示比RGB图像在仿真到真实差距上更具可迁移性，(2) 想象手部点云提供对视觉域偏移不变的几何感知表示，(3) 基于接触的奖励鼓励物理有意义的抓取行为。

物体数量： 类别内多个物体；在真实世界中展示同类别新颖物体的泛化。

任务： 多样物体的灵巧抓取。展示类别级泛化：在一组物体上训练的策略泛化至真实世界中同类别新颖物体。

核心方法论： 提出使用点云观测进行灵巧操作RL，解决困扰基于图像方法的仿真到真实差距。两个关键创新：(1) 用"想象手部点云"增强物体点云——从本体感觉状态生成的机器人手合成点云——提供手-物体空间关系的几何感知而不依赖视觉渲染。(2) 基于接触的奖励函数鼓励策略学习物理有意义的交互，而非利用仿真器伪影。点云表示自然比RGB图像更具仿真到真实可迁移性，因为它直接捕捉几何。

主要贡献：

  • 首个基于RL的策略实现灵巧抓取的类别级泛化并带仿真到真实迁移
  • 引入想象手部点云作为桥接灵巧手仿真到真实的增强表示
  • 物理有意义抓取行为的基于接触的奖励设计
  • 在Allegro Hand上跨多样物体展示零样本仿真到真实迁移

定量结果：

指标 数值
新颖物体泛化 类别级（灵巧策略学习首次）
真实世界手部 Allegro Hand（16自由度）
仿真到真实 零样本迁移
观测 点云（物体 + 想象手部）
仿真平台 IsaacGym

局限性/差距：

  • 力控： 无力传感或阻抗控制；仅位置抓取
  • VLA/语言： 无语言条件或VLA集成
  • 仿真到真实： 点云需要深度相机；深度传感质量影响真实世界性能
  • 代码： GitHub 可用
  • 任务范围： 仅限抓取；不涉及手内操作、工具使用或多阶段任务
  • 物体多样性： 展示了类别级泛化但限于有限类别；未在UniDexGrasp那样的数千物体上测试

推理 / 部署

  • 推理延迟： 未明确报告。带点云（想象手部）输入的MLP策略每次前向传播 <5ms。
  • 部署硬件： Allegro Hand（16自由度）用于真实世界部署。策略在IsaacGym中训练；通过点云观测零样本仿真到真实迁移部署。
  • 支持实时？ 是。带点云编码器的轻量策略支持真实Allegro Hand上的实时灵巧抓取。
