7.4 CyberDemo

完整标题： CyberDemo: Augmenting Simulated Human Demonstration for Real-World Dexterous Manipulation

作者： Jun Wang, Yuzhe Qin, Kaiming Kuang, Yigit Korkmaz, Akhilan Gurumoorthy, Hao Su, Xiaolong Wang

发表venue/年份： CVPR 2024

arXiv： https://arxiv.org/abs/2402.14795

RL算法： 模仿学习（行为克隆），从增强的仿真演示。使用重定向到仿真中机器人手的人手演示，然后进行广泛的域随机化增强。

手部硬件： Allegro Hand（16自由度）用于真实世界部署

仿真平台： IsaacGym / SAPIEN（用于仿真环境和演示增强）

仿真到真实？ 是。核心贡献是通过增强仿真演示实现仿真到真实迁移。演示从人手运动收集，在仿真中重定向到Allegro Hand，然后通过物体姿态、光照、物理属性和相机视角的变化进行增强。在增强仿真数据上训练的策略迁移到真实Allegro Hand。

任务： 灵巧操作包括阀门旋转（三阀、四阀、五阀）和其他物体操作任务。在不同几何和物理属性的物体上测试，包括训练期间未见的新颖物体。

核心方法论： 通过利用仿真进行数据增强来解决真实世界灵巧操作的数据瓶颈。(1) 收集操作任务的人类演示。(2) 在仿真中将人手运动重定向到Allegro Hand。(3) 通过变化物体几何、纹理、姿态、物理参数和相机视角在仿真中大规模增强演示。(4) 在增强数据集上训练视觉运动策略（行为克隆）。关键发现是增强仿真演示在策略训练中优于真实世界演示，因为仿真能实现比真实数据收集多数个数量级的多样性。

主要贡献：

  • 展示增强仿真演示在灵巧操作策略训练中超越真实世界演示
  • 仿真中的广泛数据增强流程（几何、纹理、姿态、物理、视角变化）
  • 在Allegro Hand上成功实现阀门旋转和操作任务的仿真到真实迁移
  • 泛化至训练期间未见的新颖物体，包括不同阀门配置

局限性/差距：

  • 力控： 无力传感或阻抗控制；基于位置的模仿学习
  • VLA/语言： 无语言条件或VLA骨干
  • 仿真到真实： 增强有帮助但接触动力学的仿真到真实差距仍存在；高度接触敏感任务的性能未广泛评估
  • 代码： GitHub 可用
  • 可扩展性： 需要人类演示收集作为种子；增强是自动化的但种子收集是手动的

推理 / 部署

  • 推理延迟： 未明确报告。行为克隆策略（MLP/CNN）每次前向传播 <5ms。
  • 部署硬件： Allegro Hand（16自由度）用于真实世界部署。策略在IsaacGym/SAPIEN中增强仿真演示上训练；通过零样本仿真到真实迁移部署。
  • 支持实时？ 是。轻量模仿学习策略支持真实Allegro Hand上的实时控制。
