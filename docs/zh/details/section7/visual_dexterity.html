7.2 Visual Dexterity

完整标题： Visual Dexterity: In-Hand Reorientation of Novel Objects Using Depth Sensing

作者： Tao Chen, Megha Tippur, Siyang Wu, Vikash Kumar, Edward Adelson, Pulkit Agrawal

发表venue/年份： Science Robotics 2023（原始arXiv 2022）

RL算法： PPO + 非对称Actor-Critic；教师使用特权状态，学生使用域随机化和深度图像观测训练

手部硬件： D'Claw手（9-12自由度，取决于配置）；低成本三指手

仿真平台： IsaacGym Preview 3

仿真到真实？ 是；使用域随机化（视觉和物理）的零样本仿真到真实迁移（真实）。仅深度感知通过绕过RGB仿真到真实差距实现鲁棒迁移

物体数量： 在真实世界实验中评估了100+训练期间未见的新颖物体

任务： 到目标姿态的任意手内重定向（SO(3)重定向，不仅是单轴旋转）

核心方法论： 使用腕部安装的RealSense相机的深度图像训练视觉运动策略，用于手内重定向期间的6D物体姿态追踪。关键架构选择是仅深度输入，其仿真到真实差距远小于RGB。非对称Actor-Critic在仿真中使用特权物体姿态训练；部署策略仅使用深度观测。广泛的程序化物体生成创建了多样的训练物体。

主要贡献：

  • 在真实硬件上使用新颖物体实现完整SO(3)手内重定向（不仅是单轴旋转）（真实）
  • 仅深度感知以实现最小仿真到真实视觉差距
  • 泛化至100+新颖真实世界物体，包括不在训练分布中的家用物品（真实）

局限性/差距： D'Claw是三指手，相比五指手灵巧性有限；深度相机对透明/反光物体有局限；仅重定向（无平移控制）；需要腕部安装相机且有清晰物体视野

推理 / 部署

  • 推理延迟： 未明确报告。带深度图像输入的MLP策略每次前向传播 <5ms（深度图像编码 + MLP）。
  • 部署硬件： D'Claw手（9-12自由度）+ 腕部安装Intel RealSense深度相机。策略在IsaacGym中训练；通过域随机化零样本仿真到真实迁移部署。
  • 支持实时？ 是。轻量策略支持实时控制。已在真实D'Claw手上使用100+新颖物体验证。

数据集 / 数据收集

  • 使用的数据集： 无预收集数据集。纯RL（PPO）+ 非对称Actor-Critic——所有数据在仿真中生成。
  • 收集方法： IsaacGym中的纯RL，带域随机化（视觉和物理）。教师使用特权物体状态；学生使用腕部RealSense相机的深度图像观测训练。广泛的程序化物体生成创建多样训练物体。零样本仿真到真实迁移。
  • 数据规模： 程序化生成训练物体。100+新颖真实世界物体用于评估（训练期间未见）。
  • 遥操作设备： 不适用（纯RL，无演示）。
  • 数据格式： 不适用（在线RL，无离线数据集）。
  • 是否公开？ 代码发布状态未报告。
