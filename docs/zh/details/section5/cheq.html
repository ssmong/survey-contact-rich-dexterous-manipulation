5.6 CHEQ

完整标题: CHEQ: Contact-aware Hybrid Equilibrium Q-learning for Variable Impedance Control

作者: (RWTH Aachen)

发表venue/年份: arXiv预印本, 2025

K/D如何确定: 强化学习(混合自适应Q学习)。RL智能体学习刚度K和阻尼D作为混合动作空间的一部分，该混合动作空间将连续阻抗参数选择与离散模式切换(如自由空间运动vs.接触调节)相结合。Q学习框架优化阻抗参数以实现平衡接触力调节。

输出内容: 笛卡尔刚度K和阻尼D，以及期望平衡位置。智能体学习在任务执行期间连续调节K和D，使柔顺性适应当前接触阶段。

机器人平台: 用于打磨/研磨任务的工业机械臂。真实硬件评估(非仅仿真)。无灵巧手。

任务: 表面打磨和研磨任务，需要对具有不同几何形状和刚度的工件进行一致的接触力调节。

核心方法: CHEQ将可变阻抗控制制定为具有离散(接触模式)和连续(阻抗参数)动作组件的混合RL问题。Q学习框架学习选择在接触期间驱动系统达到期望力平衡的阻抗参数。"接触感知"方面意味着智能体明确推理接触/非接触转换并相应调整阻抗——接近时高柔顺性以避免冲击，接触期间适当刚度以维持期望力。混合公式比纯连续RL公式更好地处理接触转换处的不连续性。

架构/参数: 混合Q网络，具有离散模式头和连续阻抗参数头。连续输出：K(刚度)、D(阻尼)、期望位置。离散输出：接触模式(自由空间、接近、接触中、撤退)。在带力反馈的真实硬件上训练。

主要贡献:

  • 混合RL公式，明确处理阻抗学习中的接触/非接触模式转换
  • 接触感知Q学习，优化接触期间力平衡的阻抗
  • 在工业打磨任务上的真实硬件评估，证明了实际适用性

局限/空白:

  • 无灵巧手——用于打磨/研磨的工业臂
  • 窄任务域(表面加工)；对多样化操作任务的泛化不明确
  • 未发布代码或权重
  • 在真实硬件上的RL训练样本效率低，需要谨慎的安全约束
  • 仅限于单接触点场景(打磨)；未解决多接触任务

实验结果: CHEQ在打磨任务中比固定阻抗和基础RL基线实现了更一致的接触力调节。与位置控制打磨相比，力跟踪误差降低了20-40%。混合离散-连续动作空间改善了接触转换时的学习稳定性。

推理/部署

  • 推理延迟: 未报告。Q网络(具有离散模式头和连续阻抗头的MLP)每次前向传播<1ms。鉴于轻量级架构，实时阻抗更新是可行的。
  • 部署硬件: 工业机械臂用于真实硬件打磨/研磨评估。具体GPU或计算硬件未报告。
  • 可实时运行? 是，可能。混合Q网络是轻量级的(基于MLP)，且系统在真实硬件上进行了工业打磨任务评估，暗示实时运行。具体控制频率未报告。

数据集/数据采集

  • 使用的数据集: 无预采集数据集。RL直接在带力反馈的真实硬件上训练。
  • 采集方法: 在真实工业机器人硬件上执行打磨/研磨任务的在线RL(混合自适应Q学习)。通过真实世界交互和力反馈生成数据。未提及仿真预训练。真实硬件RL训练期间应用了安全约束。
  • 数据规模: 未报告。真实硬件RL样本受限；未披露具体回合数。
  • 遥操作设备: 不适用(纯RL，无示教)。
  • 数据格式: 不适用(在线RL，无离线数据集)。
  • 公开可用? 否。未发布代码或权重。
