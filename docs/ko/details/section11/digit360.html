# 11.5 Digit 360

  • 전체 제목: Digit 360: A Multimodal Tactile Fingertip Sensor
  • 저자: Meta FAIR
  • 학회/연도: 2024
  • 유형: Tactile 센서 (손가락 끝 모듈), 완전한 hand 아님
  • 센싱 모달리티: 법선력/전단력, 진동, 온도, 근접, 고해상도 tactile 이미지 등 18개 이상 모달리티
  • 비용: 미공개
  • 통합 대상: 임의의 dexterous hand 플랫폼 (모듈식 손가락 끝으로 설계)

정량 사양

사양 값
센싱 모달리티 18개 이상 동시 (법선력, 전단력, 진동, 온도, 근접, tactile 이미지 등)
형태 손가락 끝 크기, 기존 hand에 직접 교체 가능하도록 설계
계산 오버헤드 상당함 -- 18개 이상 모달리티의 병렬 데이터 처리 필요
가용성 아직 공개적으로 이용 불가 (서베이 시점 기준 오픈소스 공개 계획)

핵심 방법론/설계

Digit 360은 hand가 아니라 dexterous hand에 통합되도록 설계된 멀티모달 tactile 손가락 끝 센서이다. Meta의 DIGIT 계열 비전 기반 tactile 센서를 확장하여, 단일 카메라 기반 tactile 이미지에서 공간적으로 분산된 법선력 및 전단력, 고주파 진동, 온도, 근접 센싱을 포함하는 18개 이상 동시 모달리티로 센싱 모달리티를 극적으로 확장한다. 센서는 기존 dexterous hand의 표준 손가락 끝을 교체하도록 설계된 소형 형태를 사용한다.

주요 기여

  • 18개 이상 동시 센싱 모달리티를 갖춘 가장 포괄적인 tactile 손가락 끝 센서 중 하나
  • 단일 손가락 끝 크기 패키지에 멀티모달 센싱(힘, 진동, 온도, 근접)
  • 모듈식 업그레이드로 기존 dexterous hand 플랫폼에 통합되도록 설계
  • 현재 저비용 hand(LEAP, RUKA, ISyHand 모두 tactile 부재)의 센싱 격차 해소

한계점

  • 완전한 hand가 아님; 기존 hand 플랫폼에 통합 필요
  • 비용 미공개; hand 자체보다 상당히 비쌀 가능성
  • 아직 연구 커뮤니티에 널리 이용 가능하지 않음 (오픈소스 공개 예정)
  • 18개 이상 모달리티의 동시 데이터 처리에 계산 오버헤드 발생
  • tactile 장착 hand를 위한 기존 sim-to-real 파이프라인과의 통합이 아직 시연되지 않음
