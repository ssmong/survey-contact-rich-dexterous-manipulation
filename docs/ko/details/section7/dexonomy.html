7.1 Dexonomy

전체 제목: Dexonomy: A Taxonomy and a Scalable Generation Pipeline for Dexterous Grasping

저자: Jiayi Chen, Yuxing Chen, C. Karen Liu, He Wang

학회/연도: RSS 2025

PKU-EPIC lineage: Extends BODex with 31 grasp types (vs untyped in BODex); the bilevel optimization core is shared with BODex, but Dexonomy adds a systematic grasp taxonomy derived from human grasping literature. The taxonomy-driven contact region templates are the primary contribution beyond BODex.

RL 알고리즘: Optimization-based (not RL); extends BODex with a taxonomy-driven grasp generation pipeline; uses cuRobo for IK solving

핸드 하드웨어: Shadow Hand (24 DoF), Allegro Hand (16 DoF), LEAP Hand (16 DoF), Unitree G1 humanoid hand

시뮬레이션 플랫폼: MuJoCo + cuRobo

Sim2Real 여부: Yes; 82.3% real-world success rate (real)

객체 수: 10,700 objects across 31 grasp types; 9.5 million generated grasps

작업: Taxonomy-guided dexterous grasp generation with 31 distinct grasp types (precision, power, lateral, etc.)

핵심 방법론: Introduces a comprehensive grasp taxonomy with 31 types derived from human grasping literature, then builds a scalable pipeline that generates type-specific grasps for arbitrary objects. For each grasp type, contact region templates specify which hand surfaces contact which object surfaces. The optimization pipeline matches these templates to object geometry and solves for feasible hand configurations.

주요 기여:

  • First systematic grasp taxonomy (31 types) applied to robotic dexterous grasp generation
  • Scalable pipeline generating 9.5M grasps across 10.7K objects and 4 hand embodiments
  • Cross-embodiment generalization across Shadow, Allegro, LEAP, and Unitree G1 hands

정량적 결과:

Metric Value
Real-world success rate 82.3% (real)
Object set size 10,700
Grasp types 31
Generated grasps 9.5M
Hands supported Shadow, Allegro, LEAP, Unitree G1

한계점: Taxonomy is static and may not capture task-specific grasp requirements; optimization-based (no learned closed-loop policy); real-world validation limited to a subset of grasp types; assumes known object geometry

추론 / 배포

  • 추론 지연 시간: Optimization-based grasp computation is offline. cuRobo-accelerated IK solving. Deployment executes pre-computed grasps open-loop.
  • 배포 하드웨어: Real deployment on multiple hand platforms (Shadow, Allegro, LEAP, Unitree G1). 82.3% real-world success rate.
  • 실시간 가능 여부: Yes, at deployment (pre-computed grasps executed open-loop). No online inference bottleneck.

데이터셋 / 데이터 수집

  • 사용 데이터셋: Procedurally generated taxonomy-driven grasp dataset. 10,700 objects, 31 grasp types, 9.5 million grasps total.
  • 수집 방법: Optimization-based (extends BODex pipeline with taxonomy-driven contact region templates). cuRobo for IK solving. Cross-embodiment: Shadow, Allegro, LEAP, Unitree G1 hands. No RL, no demonstrations.
  • 데이터 규모: 9.5 million generated grasps across 10,700 objects and 4 hand embodiments. 31 grasp types (precision, power, lateral, etc.).
  • 원격 조작 장비: Not applicable (optimization-based, no demonstrations).
  • 데이터 포맷: Grasp configurations indexed by object, hand, and grasp type.
  • 공개 여부: Dataset release status not reported. CAD and code availability not specified.
