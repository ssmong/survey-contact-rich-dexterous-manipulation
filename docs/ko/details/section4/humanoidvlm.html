4.3 HumanoidVLM

전체 제목: VLM-based Impedance Control for Safe Physical Human-Robot Interaction with Humanoid Robots

저자: (다수 저자, SafeHumanoid 그룹과 중복 가능)

학회/연도: HRI 2026

K/D 결정 방법: VLM (Molmo-7B, 오픈 가중치) + FAISS 기반 RAG 검색. 작업 맥락과 시각 장면 특징으로 인덱싱된 이전 임피던스 구성의 벡터 데이터베이스 사용. 새 상호작용 시나리오에서 VLM이 FAISS 데이터베이스에서 유사한 이전 구성을 검색하고 현재 맥락에 적응. Molmo-7B(7B 파라미터 오픈 가중치 VLM)가 추론을 수행하여 상용 API 의존 회피.

출력: 휴머노이드 엔드이펙터 임피던스 컨트롤러를 위한 검색 및 적응된 직교 강성 K 및 감쇠 D 파라미터.

로봇 플랫폼: Unitree G1 휴머노이드 로봇. 독립 정교한 핸드 없음(G1의 통합 Dex3-1 그리퍼, 핸드당 7 DoF 사용).

작업: 물리적 인간-로봇 상호작용(pHRI) 시나리오: 협동 물체 전달, 안내 조작, 물리적 보조 작업. 휴머노이드가 인간 파트너 부상을 방지하기 위해 유연해야 하는 안전 중요 상호작용에 초점.

핵심 방법론: 상호작용 맥락의 의미적 이해를 위한 VLM과 적절한 임피던스 파라미터의 RAG 기반 검색을 결합. Molmo-7B가 시각 장면(인간 파트너의 자세와 의도 단서 포함)을 처리하고 유사한 이전 상호작용에서 성공적이었던 임피던스 구성을 FAISS 데이터베이스에서 검색. 상용 API 대신 오픈 가중치 VLM(Molmo-7B) 사용이 CompliantVLA-adaptor 및 OmniVIC와 차별화.

주요 기여:

  • VLM 안내 임피던스 제어를 물리적 인간-로봇 상호작용을 위한 휴머노이드 로봇에 적용
  • 상용 API 대신 오픈 가중치 VLM(Molmo-7B) 사용으로 온디바이스 배포 가능
  • 전체 휴머노이드 플랫폼에서 안전 중요 pHRI 시나리오를 위한 RAG 기반 임피던스 검색 시연

한계점:

  • 독립 정교한 핸드 없음 -- G1의 통합 단순 그리퍼에 의존
  • 접촉 중 연속 조절이 아닌 작업 수준 임피던스 파라미터 검색/적응
  • VLM 추론 지연(~100ms-1s)이 임피던스 업데이트를 ~1-10 Hz로 제한

결과: Unitree G1 휴머노이드에서 안전한 물리적 상호작용 시연, VLM 안내 임피던스 컨트롤러가 다양한 상호작용 유형에 적절한 유연성 수준 생성. 전달 및 협동 조작 작업 중 고정 임피던스 기준선 대비 최대 상호작용 힘 감소.

추론 / 배포

  • 추론 지연 시간: 명시적 벤치마크 없음. Molmo-7B 추론(오픈 가중치, 온디바이스)은 현대 GPU에서 쿼리당 ~200-500ms 소요 가능, FAISS 검색 시간(<10ms) 추가.
  • 배포 하드웨어: Unitree G1 휴머노이드. Molmo-7B는 로컬 GPU에서 실행 가능(구체적 모델 미보고), 클라우드 API 의존 회피.
  • 실시간 가능 여부: 임피던스 적응에 대해 아니오. 오픈 가중치 모델 사용이 온디바이스 배포를 가능하게 하나 실시간 연속 임피던스 조절은 달성하지 못함.

데이터셋 / 데이터 수집

  • 사용 데이터셋: pHRI 시나리오를 위한 맞춤 FAISS 기반 RAG 임피던스 구성 데이터베이스.
  • 수집 방법: Unitree G1에서의 이전 pHRI 시행의 초기 임피던스 구성으로 수동 구축된 RAG 데이터베이스.
  • 공개 여부: 보고되지 않음. HRI 2026 학회; 공개 상태 미상.
