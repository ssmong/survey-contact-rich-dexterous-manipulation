3.1 ForceVLA

  • 전체 제목: ForceVLA: Towards Force-Aware Vision-Language-Action Model for Contact-Rich Manipulation
  • 저자: Qiaojun Yu et al. (SJTU, Fudan University)
  • 학회/연도: NeurIPS 2025
  • arXiv: https://arxiv.org/abs/2505.22159

힘/촉각 입력 유형: Flexiv Rizon 팔 손목에 장착된 6축 힘/토크(F/T) 센서. 원시 F/T 신호(Fx, Fy, Fz, Tx, Ty, Tz)가 토큰화되어 시각 토큰과 함께 VLA 백본에 입력된다.

힘/임피던스 출력: 없음. 위치 전용 출력. 힘 정보는 정책 결정을 개선하기 위한 입력 맥락으로 사용되나, 로봇은 Flexiv Rizon의 내장 컨트롤러를 통해 위치 명령을 실행한다.

한계 (위치 전용 출력): 위치 전용 출력; 힘/촉각 입력을 수신함에도 정책이 접촉 힘을 능동적으로 조절할 수 없다.

로봇 플랫폼: Flexiv Rizon 4 로봇 팔 + 병렬 조 그리퍼.

한계 (그리퍼 전용): 병렬 조 그리퍼에서만 평가; 다지 정교한 핸드에서는 검증되지 않음.

작업: 5가지 접촉이 풍부한 조작 작업 -- 플러그 삽입, 표면 닦기, 테이프 벗기기, 힘 민감 정렬을 포함한 컵 쌓기, 지우개 닦기. 모든 작업은 작업 진행 판단이나 동작 조정을 위해 접촉 힘 해석이 필요하다.

핵심 방법론: ForceVLA는 사전 훈련된 VLA(OpenVLA 아키텍처 기반)를 연속 6축 F/T 판독값을 VLM 어휘와 호환되는 이산 토큰으로 변환하는 힘 토크나이저를 도입하여 확장한다. 힘 토큰은 변환기의 입력 시퀀스에서 비전 및 언어 토큰과 인터리빙된다. 동기화된 F/T 녹화를 포함하는 소규모 원격 조작 시연 데이터셋에서 엔드-투-엔드 미세 조정된다. 힘 조건부 어텐션 메커니즘은 다음 행동 청크 생성 시 힘 이력에 대한 어텐션을 가능하게 한다.

아키텍처/파라미터: OpenVLA (7B 파라미터) 기반. 힘 토크나이저는 6D F/T 판독값을 VLM 임베딩 공간으로 투영하는 경량 MLP 인코더를 추가. 행동 출력은 위치 기반 (7D: 6-DoF 엔드이펙터 자세 + 그리퍼).

주요 기여:

  • 6축 F/T 센싱을 VLA의 명시적 입력 모달리티로 통합하여, 연속 힘 판독값을 VLM 어휘와 호환되는 이산 토큰으로 토큰화하는 것을 제안
  • 시각 단서만으로는 불충분한 접촉이 풍부한 작업(예: 시각적 가림보다 힘 피드백으로 삽입 성공 판단)에서 힘 인식 VLA가 비전 전용 VLA를 유의하게 능가함을 시연
  • 5가지 접촉이 풍부한 작업에 걸쳐 동기화된 F/T 데이터를 포함하는 힘 주석 조작 데이터셋(ForceVLA-Data: 244개 궤적, ~14만 단계) 공개(HuggingFace에서 이용 가능)

한계점: 위치 전용 출력은 로봇이 접촉 힘을 능동적으로 조절할 수 없음 -- Flexiv 팔의 내장 유연성에 의존. 비교적 소규모 데이터셋으로 5개 작업에 제한. 단일 로봇 플랫폼(Flexiv Rizon)으로 교차 구현체 평가 없음. 힘 토큰화 접근법은 연속 힘 신호를 이산화하여 고주파 접촉 정보의 손실 가능성.

코드/데이터 공개: ForceVLA-Data 데이터셋이 HuggingFace에 공개(244개 궤적). 모델 코드(데이터와 구별되는)의 공개 여부는 독립적으로 검증해야 한다 -- 주요 공개 산출물은 데이터셋이다.

결과:

지표 값
비전 전용 VLA 대비 성공률 향상 5개 작업에 걸쳐 15-30%
최대 단일 작업 이득 플러그 삽입 (성공적 결합 감지에 힘 피드백 필수)
데이터셋 크기 244개 궤적, ~14만 단계

추론 / 배포

  • 추론 지연 시간: 보고되지 않음. 논문에서 단계별 추론 지연이나 제어 주파수를 공개하지 않음. pi-zero 프레임워크 기반 flow 기반 행동 생성.
  • 배포 하드웨어: 훈련에 8x NVIDIA RTX 4090 GPU. 실세계 배포를 위한 Flexiv Rizon 4 팔 + 병렬 조 그리퍼. 추론 하드웨어 별도 미명시.
  • 실시간 가능 여부: 미검증. 추론 지연과 제어 주파수가 논문이나 프로젝트 페이지에 공개되지 않음.

데이터셋 / 데이터 수집

  • 사용 데이터셋: ForceVLA-Data (본 연구와 함께 공개된 맞춤 데이터셋). 동기화된 비전, 고유감각, 힘/토크 신호로 5가지 접촉이 풍부한 조작 작업 포괄.
  • 수집 방법: 시각 및 고유감각 데이터와 함께 동기화된 6축 F/T 녹화를 포함하는 Flexiv Rizon 4 팔에서의 원격 조작 시연.
  • 데이터 규모: 5개 작업(플러그 삽입, 표면 닦기, 테이프 벗기기, 컵 쌓기, 지우개 닦기)에 걸쳐 244개 궤적, ~14만 단계.
  • 원격 조작 장비: 상세히 기술되지 않음. F/T 센서가 장착된 Flexiv Rizon 4에서의 원격 조작.
  • 데이터 포맷: 동기화된 다중 모달 데이터 (비전 + 고유감각 + 6축 F/T). 구체적 파일 포맷 미보고.
  • 공개 여부: 예. ForceVLA-Data 데이터셋이 HuggingFace에 공개(244개 궤적). 코드 및 데이터가 프로젝트 페이지(sites.google.com/view/forcevla2025)에서 이용 가능.
