3.6 DreamTacVLA

  • 전체 제목: Learning to Feel the Future: Tactile World Model for Contact-Rich VLA Policy
  • 저자: Northwestern University et al.
  • 학회/연도: arXiv 프리프린트, 2025
  • arXiv: https://arxiv.org/abs/2512.23864

힘/촉각 입력 유형: 비전 기반 촉각 센서. 촉각 이미지가 V-JEPA2(자기 지도 시각 인코더)를 통해 처리되어 원시 촉각 이미지 대신 촉각 잠재 표현 추출.

힘/임피던스 출력: 없음. 위치 전용 행동 출력. 그러나 모델 내부적으로 미래 촉각 잠재 상태를 예측(촉각 세계 모델)하여 행동 예측 개선에 사용.

로봇 플랫폼: Dobot Xtrainer 로봇 팔 + 그리퍼 + 비전 기반 촉각 센서.

작업: 촉각 피드백이 성공에 필요한 4가지 접촉이 풍부한 조작 작업.

핵심 방법론: DreamTacVLA는 현재 상태와 계획된 행동에 조건부된 미래 촉각 잠재 표현을 예측하는 촉각 세계 모델을 도입. V-JEPA2 인코더가 원시 촉각 이미지를 컴팩트한 잠재 벡터로 처리. 세계 모델 모듈(변환기 기반)이 후보 행동 시퀀스가 주어졌을 때 촉각 잠재가 어떻게 발전할지 예측. 예측된 미래 촉각 상태가 행동 예측을 개선하는 데 사용 -- 정책이 "미래를 느끼고" 바람직하지 않은 접촉 상태(과도한 힘, 미끄러짐 등)로 이어질 행동을 회피할 수 있게 함.

주요 기여:

  • 미래 접촉 상태를 예측하여 반응적이 아닌 사전적 접촉이 풍부한 조작을 가능하게 하는 촉각 세계 모델 도입
  • V-JEPA2를 촉각 인코더로 사용하여, 사전 훈련된 시각 SSL 모델이 촉각 이미지 정보를 효과적으로 인코딩할 수 있음을 시연
  • 미래 촉각 상태 예측과 이에 조건부된 행동이 촉각 미인식 및 반응 전용 촉각 접근법 대비 접촉이 풍부한 작업에서 성공률을 향상시킴을 보여줌

한계점: 코드만 공개(GitHub 저장소), 모델 가중치 미공개. 촉각 세계 모델이 추론에 계산 오버헤드 추가. 단일 로봇 플랫폼에서 4개 작업으로 제한.

결과:

지표 값
최고 성공률 4가지 접촉이 풍부한 작업에서 최대 95%
반응 전용 촉각 대비 향상 10-20%

추론 / 배포

  • 추론 지연 시간: 제어 루프가 50 Hz(데이터 수집 주파수와 일치)로 실행. Think-Dream-Act 파이프라인이 추론 오버헤드를 추가; 저자들은 이를 인정하고 "더 빠른 단일 패스 추론을 위한 정책 증류와 적응적 드리밍"을 향후 연구로 제안.
  • 배포 하드웨어: 단일 NVIDIA GPU(모델 미명시; 2만 단계에 ~8-12시간)에서 훈련. Dobot Xtrainer 팔 + 그리퍼 + 비전 기반 촉각 센서.
  • 실시간 가능 여부: 부분적. 50 Hz 제어 속도는 유지되나, 2단계 파이프라인(세계 모델 예측 + 행동 생성)이 계산 오버헤드를 도입. 저자들이 추론 속도를 한계점으로 명시적 지적.

데이터셋 / 데이터 수집

  • 사용 데이터셋: 고충실도 디지털 트윈 시뮬레이션 데이터와 실세계 실험 데이터를 결합하는 하이브리드 대규모 데이터셋.
  • 수집 방법: 하이브리드 접근: (1) 합성 촉각 데이터 생성을 위한 디지털 트윈 시뮬레이션, (2) 비전 기반 촉각 센서(GelSight 스타일)를 사용한 실세계 실험. 50 Hz로 수집.
  • 데이터 규모: 명시적으로 보고되지 않음.
  • 공개 여부: 코드 이용 가능(GitHub 저장소 참조). 모델 가중치 미공개. 데이터셋 공개 여부 미확인.
