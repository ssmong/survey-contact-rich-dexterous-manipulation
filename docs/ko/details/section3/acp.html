3.17 ACP

참고: VLA 아님 (언어 조건부 없음). 힘 출력 방법론을 위해 Section 3에 포함.
  • 전체 제목: ACP: Adaptive Compliance Policy for Contact-Rich Manipulation with Impedance Control
  • 저자: Toyota Research Institute / Columbia University et al.
  • 학회/연도: ICRA 2025
  • arXiv: https://arxiv.org/abs/2410.09309

힘/촉각 입력 유형: 6축 F/T 센서 (ATI 산업용 고정밀 센서).

힘/임피던스 출력: 예 -- 로봇 임피던스 컨트롤러를 조절하는 스칼라 강성 파라미터 출력. 힘 출력 유형: 부분 (스칼라 강성만).

로봇 플랫폼: UR5e 로봇 팔 + 수동 도구 부착(예: 주걱, 스펀지).

작업: 항목 뒤집기(주걱 사용) 및 꽃병 닦기 -- 가변 유연성이 필요한 작업.

핵심 방법론: ACP는 위치 목표와 스칼라 강성 파라미터를 모두 출력하는 적응적 유연성 정책을 학습. 강성 파라미터가 임피던스 컨트롤러의 유연성을 동적으로 조정 -- 정밀 위치 단계에서는 더 강하게, 접촉 단계에서는 더 유연하게.

주요 기여:

  • 위치 목표와 함께 적응적 유연성 학습을 제안하여 작업 단계에 기반한 동적 강성 조정 가능
  • 단일 스칼라 강성 파라미터만으로도 고정 유연성 기준선 대비 접촉이 풍부한 작업 성능 유의 향상 시연
  • 가변 유연성이 필수적인 실용적 응용(항목 뒤집기, 꽃병 닦기) 제시

한계점: 코드/가중치 미공개. 스칼라 강성 출력은 단순화 -- 축별 강성 적응이 필요할 수 있음. 2개 작업만 평가. VLA 아님. 수동 도구 사용으로 능동적 도구 조작 일반화 제한.

결과: ACP 적응적 유연성이 고정 강성 기준선 대비 항목 뒤집기와 꽃병 닦기에서 높은 성공률 달성.

추론 / 배포

  • 추론 지연 시간: 보고되지 않음. 센서/액추에이터 속도(GoPro 60 Hz, 로봇 직교 명령 500 Hz, ATI F/T 센서 최대 7000 Hz) 명시하나 정책 추론 지연 미공개.
  • 배포 하드웨어: UR5e 팔 + ATI F/T 센서 + GoPro 카메라.

데이터셋 / 데이터 수집

  • 수집 방법: UR5e 팔에서의 인간 시연. ATI F/T 센서로 기록된 시연자의 가변 파지 힘이 원하는 유연성의 프록시로 사용.
  • 공개 여부: 아니오.
