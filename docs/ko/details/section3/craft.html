3.11 CRAFT

  • 전체 제목: CRAFT: Contact-Rich Affordance-aware Force-guided Transformer for Manipulation
  • 학회/연도: arXiv 프리프린트, 2026
  • arXiv: https://arxiv.org/abs/2602.12532

힘/촉각 입력 유형: 힘 센싱.

힘/임피던스 출력: 없음. 위치 전용 출력.

로봇 플랫폼: 원격 조작 로봇 팔.

작업: 접촉이 풍부한 상호작용이 필요한 변형 가능 물체 조작 및 정렬 작업.

핵심 방법론: CRAFT는 변환기 기반 정책 내에서 힘 정보와 접촉 어포던스 추론을 통합. 접촉 어포던스 예측 모듈이 작업 관련 접촉 영역을 식별하고, 힘 안내 변환기가 원하는 접촉 상태를 달성하는 행동을 생성.

주요 기여:

  • 접촉 어포던스 예측과 힘 안내 행동 생성을 결합
  • 접촉 기하학이 동적으로 변하는 변형 가능 물체 조작 대응

한계점: 코드/가중치 미공개. 구체적 힘 센서 및 로봇 플랫폼 세부 사항 제한. 변형 및 정렬 작업으로 제한 평가.

결과: 정량적 결과 미공개. 변형 가능 물체 조작에서 힘 미인식 기준선 대비 개선 보고.

추론 / 배포

  • 추론 지연 시간: 보고되지 않음. VIB 모듈은 "경량이고 모델 비독립적"으로 기술.
  • 배포 하드웨어: 실세계 실험용 Franka Emika Panda 로봇 팔.
  • 실시간 가능 여부: 미검증.

데이터셋 / 데이터 수집

  • 수집 방법: "동기화된 비전, 언어, 힘 데이터를 수집하는 상동 리더-팔로워 원격 조작 시스템."
  • 데이터 포맷: 동기화된 다중 모달: 비전 + 언어 + 힘 데이터.
  • 공개 여부: 아니오.
