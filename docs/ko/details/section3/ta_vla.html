3.10 TA-VLA

  • 전체 제목: TA-VLA: Torque-Aware Vision-Language-Action Model for Contact-Rich Manipulation
  • 저자: Tsinghua AIR et al.
  • 학회/연도: CoRL 2025
  • arXiv: https://arxiv.org/abs/2509.07962

힘/촉각 입력 유형: 모터 전류에서 파생된 관절 토크. 추가 하드웨어 불필요.

힘/임피던스 출력: 보조 토크 예측 헤드. 훈련 정규화기 역할, 추론 시 이상 감지 선택적 사용.

로봇 플랫폼: Cobot Magic ALOHA 양팔 플랫폼.

작업: 버튼 누르기, 충전기 삽입, USB 삽입 등 10가지 접촉이 풍부한 작업.

핵심 방법론: 관절 토크를 추가 센서 없는 힘 모달리티로 사용. 다중 작업 학습으로 보조 토크 예측이 행동 예측 개선.

주요 기여:

  • 관절 토크(모터 전류)가 VLA의 효과적 힘 모달리티임을 시연
  • 보조 토크 예측이 추론 시 토크 절제 시에도 5-15% 향상
  • 10가지 작업에서 평가

한계점: 코드/가중치 미공개. 관절 토크는 전용 F/T 센서보다 노이즈 많고 해상도 낮음.

결과:

지표 값
비전 전용 VLA 대비 향상 10개 작업 이득, 접촉이 풍부한 작업에서 최대
보조 토크 예측 이점 추론 시 토크 절제에도 5-15% 향상

추론 / 배포

  • 추론 지연 시간: 행동 생성당 ~94 ms. 토크 통합 오버헤드 ~3.26 ms.
  • 배포 하드웨어: NVIDIA RTX 4090 GPU. Cobot Magic ALOHA.
  • 실시간 가능 여부: 예, ~10 Hz.

데이터셋 / 데이터 수집

  • 수집 방법: Cobot Magic ALOHA 리더 팔(양방향 원격 조작)에서의 시연.
  • 공개 여부: 아니오.
