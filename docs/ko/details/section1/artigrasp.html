ArtiGrasp

전체 제목: ArtiGrasp: Physically Plausible Synthesis of Bi-Manual Dexterous Grasping and Articulation

저자: Zijian Dong et al. (ETH Zurich)

학회/연도: 3DV 2024

핸드 하드웨어: MANO 핸드 모델 (인간 손 프록시). ArtiGrasp은 시뮬레이션에서 양손 정교한 파지 및 관절 물체 조작을 연구하기 위한 물리 기반 프록시로 인간 손의 파라메트릭 MANO 표현을 사용한다.

작업: ARCTIC 유사 데이터셋에서 8가지 양손 파지 + 관절 물체 조작 환경 (가위, 캡슐 머신, 전화기 3개 물체 제외). 작업은 두 손으로 관절 물체를 동시에 잡고 자유도를 조작하는 것을 요구한다 (예: 노트북 열기, 스테이플러 작동). 745개의 손 자세 참조가 모든 물체에 걸쳐 생성되었다.

핵심 방법론: ArtiGrasp은 물리 시뮬레이션에서 강화학습을 사용하여 물리적으로 타당한 양손 파지 및 관절 물체 조작 동작을 합성한다. 이 접근법은 두 MANO 핸드를 제어하여 관절 물체의 협조된 양손 조작을 수행하는 정책을 훈련하며, 접촉 인식 시뮬레이션을 통해 물리적 타당성을 보장한다. RaiSim 물리 엔진은 손-물체 상호작용에 대한 정확한 접촉 역학을 제공한다. 이 방법은 파지 안정성과 관절 물체 조작 진행을 공동으로 최적화하는 것을 학습한다.

아키텍처/파라미터: PPO로 훈련된 RL 정책. 사전 훈련된 모델 가중치가 공개 배포되었다. 정책은 물체 상태와 손 고유감각을 입력으로 받아 두 MANO 핸드의 관절 각도 목표를 출력한다.

시뮬레이션 플랫폼: RaiSim 물리 엔진. Sim-to-real 전이는 시연되지 않았다 -- 이 연구는 특정 로봇 핸드가 아닌 MANO 인간 손 모델을 사용하여, 로봇 배포보다는 동작 합성에 초점을 맞춘다.

주요 기여:

  • 단일 정책에서 양손 파지와 관절 물체 조작을 공동으로 합성한다. 이전 연구는 파지와 관절 물체 조작을 별개의 문제로 다루었으나, 이 연구는 파지 안정성과 관절 물체 조작 진행을 동시에 최적화한 최초의 연구이다.
  • RaiSim에서의 물리 기반 RL을 통해 8개 관절 물체에 걸쳐 물리적으로 타당한 양손 동작을 시연하며, 운동학적 보간에 의존하지 않고 접촉 역학과 관절 한계를 준수하는 동작을 생성한다.
  • 사전 훈련된 모델과 코드를 공개한다 (GitHub), 재현성을 보장한다.

한계점:

  • 힘/임피던스 인식: 없음. RaiSim 물리 엔진이 계산하는 암묵적 접촉 힘 이외의 명시적 힘 센싱이나 임피던스 제어가 없다. 정책은 관절 위치 목표를 출력하며, 접촉 힘은 명시적으로 제어되는 것이 아니라 시뮬레이션의 접촉 솔버에서 발생한다. 이는 파지 힘 크기(예: 깨지기 쉬운 물체 파손 방지)나 관절 조작 힘(예: 관절 한계 감지)을 시스템이 조절할 수 없음을 의미한다.
  • VLA/언어 조건부: 없음. 언어 조건부나 비전-언어 통합이 없다.
  • Sim-to-real: 현재 형태에서는 해당 없음. 특정 로봇 핸드가 아닌 MANO 인간 손 모델을 사용하여, 직접적 로봇 배포가 용이하지 않다 -- MANO에서 물리 로봇 핸드로의 리타게팅과 추가 sim-to-real 격차 해소가 필요하다.
  • 코드/데이터 공개: 코드 및 사전 훈련된 모델 가중치 공개 (GitHub).

결과: 8개 관절 물체(745개 손 자세 참조)에서 평가. 지표는 관절 조작 성공률, 변위(물체 기저부 이동), 위치 오차, 관절 각도 오차를 포함한다. 결과는 물체 전체에 걸쳐 집계된다.

Table 2 -- 파지 및 관절 조작 (결합, 물체 전체 집계):

지표 ArtiGrasp (본 연구) D-Grasp (기준선)
관절 조작 성공률 0.55 0.22
물체 기저부 변위 0.01 m 0.49 m

Table 3 -- 동적 물체 파지 및 관절 조작:

지표 ArtiGrasp (본 연구) D-Grasp (기준선)
작업 성공률 0.50 0.11
위치 오차 0.03 m 0.05 m
관절 각도 오차 0.41 rad 0.66 rad

Table 4 -- 노이즈가 있는 손 자세 추정(이미지 기반 입력)에 대한 강건성:

지표 MoCap 입력 노이즈 (이미지 기반)
파지 성공 0.67 0.64

ArtiGrasp은 D-Grasp 대비 2.5배 높은 관절 조작 성공률과 4.5배 높은 동적 작업 성공률을 달성하며, 거의 0에 가까운 물체 변위(0.01m vs. 0.49m)를 보인다.

추론 / 배포

  • 추론 지연 시간: 보고되지 않음. RaiSim 시뮬레이션에서 MANO 인간 손 모델 사용; 물리 로봇 배포 없음.
  • 배포 하드웨어: 시뮬레이션 전용 (RaiSim 물리 엔진). 실세계 배포 하드웨어 없음.
  • 실시간 가능 여부: 해당 없음 (시뮬레이션 전용, 물리 로봇 핸드가 아닌 MANO 핸드 모델).

데이터셋 / 데이터 수집

  • 사용 데이터셋: 외부 데이터셋 없음. 훈련 데이터는 6개 관절 물체 환경에서의 RaiSim RL 시뮬레이션 롤아웃을 통해 생성. 손 자세 참조는 정책을 안내하는 데 사용.
  • 수집 방법: RaiSim 물리 엔진에서 PPO를 사용한 RL 시뮬레이션 롤아웃. 정책은 물리 기반 RL을 통해 양손 파지 및 관절 조작을 처음부터 학습. 평가를 위해 기성 회귀 모델의 이미지 기반 손-물체 자세 추정이 사용됨.
  • 데이터 규모: 보고되지 않음 (on-policy RL 훈련; 데이터가 실시간 생성).
  • 원격 조작 장비: 해당 없음 (RL 훈련, 시연 없음).
  • 데이터 포맷: 해당 없음 (on-policy RL 훈련). 사전 훈련된 모델 가중치 공개.
  • 공개 여부: 예. 코드 및 사전 훈련된 모델 가중치 공개 (GitHub).
