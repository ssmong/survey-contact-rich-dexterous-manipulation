DexterityGen

전체 제목: DexterityGen: Foundation Controller for Unprecedented Dexterity

저자: UC Berkeley / Meta et al. 연구팀

학회/연도: RSS 2025

핸드 하드웨어: 로봇 팔에 장착된 Allegro Hand (16 DoF). Allegro가 도구 사용 정교성 평가의 주요 플랫폼으로 사용된다.

작업: 펜 조작(쓰기/그리기), 스크류드라이버 작동, 주사기 취급을 포함하는 정교한 도구 사용 작업. 이 작업들은 도구 매개 행동을 위한 정밀한 손끝 제어와 조율된 다지 동작을 요구한다.

핵심 방법론: DexterityGen은 도구 사용 작업에 걸쳐 일반화되는 정교한 조작을 위한 기반 컨트롤러(3가지 도구 시연)를 훈련한다. IsaacGym에서 강화학습을 사용하여 특정 도구 사용 시나리오에 적응할 수 있는 범용 정교한 컨트롤러를 개발하는 접근법이다. 기반 컨트롤러 개념은 새로운 도구와 작업에 빠르게 특화될 수 있는 충분한 손가락 조율 기술을 갖춘 기본 정책을 제공하여 작업별 훈련 부담을 줄이는 것을 목표로 한다.

아키텍처/파라미터: IsaacGym에서 대규모 병렬 환경으로 훈련된 RL 기반 정책. 기반 컨트롤러 아키텍처는 새로운 도구 사용 작업으로의 전이를 가능하게 한다. 공개 코드나 가중치 없음.

시뮬레이션 플랫폼: NVIDIA IsaacGym. Sim-to-real 전이가 시연됨 -- 시뮬레이션에서 훈련된 정책의 실세계 배포 성공을 보여준다.

주요 기여:

  • 도구 사용 작업에 걸쳐 전이 가능한 재사용 가능 기본 정책을 제공하는 "기반 컨트롤러" 개념(3가지 도구 시연: 펜, 스크류드라이버, 주사기)을 도입하여, 각 도구 사용 정책을 처음부터 훈련하는 것 대비 작업별 훈련 시간을 단축한다.
  • 정밀하고 다양한 손끝 제어 전략을 요구하는 Allegro Hand에서의 정교한 도구 사용을 시연한다 -- 펜 쓰기는 미세한 팁 제어, 스크류드라이버 회전은 토크 생성 손가락 조율, 주사기 취급은 선형 밀기 동작을 요구한다.
  • 도구 사용 작업에 대한 sim-to-real 전이를 달성하여, 기반 컨트롤러 개념이 물리적 Allegro Hand 시스템에서 sim-to-real 격차를 극복함을 보여준다.

한계점:

  • 힘/임피던스 인식: 없음. 힘/임피던스 제어 사용 없음. 도구-환경 접촉 힘(예: 종이 위 펜 압력, 스크류드라이버 삽입 토크)이 명시적으로 조절되지 않는다. 시스템은 학습된 위치 궤적과 암묵적 유연성에 의존하여 적절한 접촉 힘을 생성하며, 이는 정밀한 힘 조절이 필요한 작업에 대한 적용 가능성을 제한할 수 있다.
  • VLA/언어 조건부: 없음. 언어 조건부나 VLA 통합 없음. 기반 컨트롤러는 언어 지정 작업 설명이 아닌 RL 미세 조정을 통해 새 도구에 적응한다.
  • Sim-to-real: 예. IsaacGym에서 훈련된 정책이 물리적 Allegro Hand에 성공적으로 배포됨.
  • 코드/데이터 공개: 코드 또는 가중치 공개 없음.

결과: 원자적 정교한 기술과 장기 지평 도구 사용 작업에서 평가. 원시 원격 조작 기준선은 대부분의 작업에서 완전히 실패; DexterityGen은 안정성과 성공 모두를 가능하게 한다.

추론 / 배포

  • 추론 지연 시간: NVIDIA RTX 4090에서 추론 사이클 당 ~27 ms (37 Hz). 8단계 및 12단계 DDIM 샘플링 단계로 테스트되었으며, 행동 정확도와 지연 시간 사이의 트레이드오프가 있다.
  • 배포 하드웨어: Lambda 워크스테이션의 NVIDIA RTX 4090 GPU. 실세계 배포를 위해 로봇 팔에 장착된 Allegro Hand (16 DoF).
  • 실시간 가능 여부: 예. 확산 샘플링이 37 Hz로 실행; 실제 로봇 제어는 10 Hz로 작동. 인간 원격 조작은 별도 리타게팅 방법으로 300 Hz로 작동. 10 Hz 제어 속도는 팔 수준 도구 사용에는 충분하나 고속 정교한 손가락 제어에 필요한 주파수보다 낮다.

데이터셋 / 데이터 수집

  • 사용 데이터셋: 외부 데이터셋 없음; IsaacGym에서의 RL 시뮬레이션 롤아웃으로 생성된 훈련 데이터, 모방 학습을 위한 인간 원격 조작 시연으로 보완.
  • 수집 방법: 기반 컨트롤러 사전 훈련을 위한 IsaacGym RL 시뮬레이션 롤아웃; 물리적 Allegro Hand에서 300 Hz 리타게팅 방법을 통한 인간 원격 조작으로 시연 수집.
  • 데이터 규모: 보고되지 않음 (RL 훈련은 실시간 데이터 생성; 원격 조작 시연 수 미명시).
  • 원격 조작 장비: 실세계 시연 수집을 위해 300 Hz로 작동하는 인간 손 리타게팅 시스템.
  • 데이터 포맷: 보고되지 않음.
  • 공개 여부: 아니오. 코드, 데이터 또는 모델 가중치 공개 없음.
