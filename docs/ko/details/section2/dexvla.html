2.3 DexVLA

전체 제목: DexVLA: Vision-Language Model with Plug-In Diffusion Expert for Dexterous Manipulation

저자: Junjie Wen, Yichen Zhu, Jinming Li, Zhibin Tang, Chaomin Shen, Feifei Feng

학회/연도: CoRL 2025

arXiv: 2502.05855

핸드 하드웨어: Franka Emika 로봇 팔에 장착된 Inspired Dexterous Hand (6 DoF 핸드 관절). 행동 공간은 12차원: SE(3) 엔드이펙터 자세(3D 위치 + 3D 방향) + 6D 핸드 관절 공간. 교차 구현체 평가는 단일 팔 그리퍼와 양팔 UR5e 설정도 포함.

작업:

  • 커리큘럼을 통한 정교한 기술 학습
  • 다지 조작 작업
  • 정교한 핸드를 사용한 집기-놓기
  • 인핸드 조작

핵심 방법론: DexVLA는 동결된 비전-언어 모델에 10억 파라미터 확산 전문가를 부착하는 플러그인 아키텍처를 도입한다. VLM 백본은 정교한 훈련 중 동결 상태를 유지하면서 시각적 이해와 언어 그라운딩을 제공한다. 확산 전문가는 VLM의 잠재 표현에 조건부되어 고차원 정교한 핸드 행동을 생성한다. 이 설계는 정밀한 정교한 제어를 가능하게 하면서 VLM의 사전 훈련된 지식의 파괴적 망각을 방지한다. 커리큘럼 학습 전략은 작업 난이도를 점진적으로 증가시킨다.

아키텍처/파라미터:

  • VLM 백본: 동결된 사전 훈련 비전-언어 모델 (요약 수준에서 백본 크기 미공개)
  • 행동 헤드: ScaleDP (Scalable Diffusion Policy), 두 변형 제공:
  • ScaleDP-H (대형 변형)
  • ScaleDP-L (경량 변형)
  • 확산 전문가에 약 1B 파라미터
  • ScaleDP-H와 ScaleDP-L 체크포인트가 HuggingFace에 공개

주요 기여:

  • 플러그인 확산 전문가 아키텍처: 동결된 VLM에 부착되어 VLM의 사전 훈련 능력을 저하시키지 않으면서 정교한 조작을 가능하게 하는 1B 파라미터 확산 모델 (이 주장은 항목에서 독립적으로 검증되지 않음)
  • ScaleDP (Scalable Diffusion Policy): 고차원 정교한 행동 공간을 위해 설계된 확장 가능한 확산 기반 행동 생성 모듈, 대형(H)과 경량(L) 변형 모두 제공
  • 정교한 작업 복잡성을 점진적으로 증가시키는 커리큘럼 기반 훈련 전략으로 샘플 효율성과 최종 성능 향상

한계점:

  • 위치 목표만 출력; 힘, 임피던스 또는 유연성 출력 없음
  • 1B 확산 전문가는 동결된 VLM 위에 상당한 계산 오버헤드 추가
  • UniDex-VLA의 8종 핸드 평가와 달리 서로 다른 정교한 핸드 유형 간 교차 구현체 평가가 주요 초점 아님
  • 촉각이나 힘/토크 센서 통합 없음
  • 시뮬레이션 대비 실세계 평가 범위가 좁음

훈련 효율성: 확산 전문가만 훈련 시 0.89 에포크/시간 vs. 전체 VLA 0.32 에포크/시간 (2.78배 속도 향상).

추론 / 배포

  • 추론 지연 시간: 단일 NVIDIA A6000 GPU에서 60 Hz로 실행. 조사된 문헌에서 보고된 가장 빠른 VLA 추론 속도 중 하나.
  • 배포 하드웨어: 추론용 단일 NVIDIA A6000 GPU. 확산 전문가를 10억 파라미터로 확장함에도 비용 효율적 훈련과 빠른 추론 달성.
  • 실시간 가능 여부: 예. 단일 A6000에서 60 Hz는 실시간 정교한 조작 제어에 충분하며, 대부분의 VLA 기반 시스템의 일반적 제어 주파수인 10-30 Hz를 초과.

데이터셋 / 데이터 수집

  • 사용 데이터셋: 단일 팔, 양팔, 정교한 핸드 로봇을 포괄하는 교차 구현체 훈련 데이터. 단일 명명된 벤치마크 없음; 데이터 소스 조합 사용.
  • 수집 방법: 교차 구현체 데이터 수집. 구체적 수집 방법론(원격 조작 장치, 시뮬레이션 등)은 공개 자료에 상세히 기술되지 않음.
  • 데이터 규모: 이용 가능한 요약에 보고되지 않음. 구현체별 데이터셋 크기는 논문을 참조해야 함.
  • 원격 조작 장비: 보고되지 않음.
  • 데이터 포맷: 보고되지 않음.
  • 공개 여부: ScaleDP-H 및 ScaleDP-L 가중치가 HuggingFace에 공개. 코드: GitHub. 훈련 데이터 공개 여부 미확인.
