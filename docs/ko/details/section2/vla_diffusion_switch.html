2.7 VLA + Diffusion Policy Switching

전체 제목: Bi-Manual Dexterous Manipulation via Reinforcement Learning with Vision-Language Action Switching

저자: Cheng Pan, Kai Junge, Benhui Dai, Qinghua Guan, Josie Hughes (CREATE Lab, STI, EPFL, Lausanne; Embodied AI, Lausanne)

학회/연도: arXiv 프리프린트, 2024

arXiv: 2410.14022

핸드 하드웨어: ADAPT Hand (13 DoF) -- 유연한 조작을 위해 설계된 직렬 탄성, 건 구동 다지 핸드

작업:

  • 정교한 다지 핸드를 사용한 집기-놓기
  • 고수준 VLA 명령과 저수준 확산 정책 실행 간 정책 전환을 통한 VLA 안내 조작

핵심 방법론: 이 연구는 직렬 탄성 다지 핸드에서의 정교한 조작을 위한 비전-언어-행동 모델과 확산 정책 간 전환 메커니즘을 제안한다. VLA는 언어 지시에서 고수준 작업 이해와 대략적 동작 계획을 처리하고, 확산 정책은 접촉이 풍부한 단계에서 세밀한 정교한 제어를 담당한다. 전환 로직은 작업 상태에 기반하여 VLA 수준 명령과 확산 수준 실행 사이의 전환 시점을 결정한다. ADAPT Hand의 직렬 탄성 구동은 고유한 기계적 유연성을 제공하여 위치 기반 정책 출력을 보완한다.

아키텍처/파라미터:

  • 고수준: 언어 조건부 작업 이해를 위한 VLA 모델
  • 저수준: 세밀한 정교한 행동 생성을 위한 확산 정책
  • 전환 메커니즘: VLA와 확산 정책 간 상태 의존적 전환
  • 핸드: 13 DoF 및 직렬 탄성 요소를 갖춘 ADAPT Hand
  • 구체적 모델 크기는 공개되지 않음

주요 기여:

  • 의미적 이해를 위한 VLA와 정밀한 운동 제어를 위한 확산 정책 각각의 강점을 활용하는 VLA-확산 정책 전환 프레임워크 제안
  • 직렬 탄성 다지 핸드(ADAPT, 13 DoF)에서 접근법을 시연하며, 강체 위치 제어 핸드보다 유연한 정교한 핸드를 대상으로 하는 소수의 VLA 연구 중 하나
  • 접촉이 풍부한 환경에서 학습된 위치 정책의 보완으로서 기계적 유연성(직렬 탄성 구동)의 가치를 강조

한계점:

  • 오픈 소스 코드나 모델 가중치 공개 없음
  • 제한된 작업 다양성: 주로 집기-놓기; 도구 사용, 조립 또는 인핸드 조작은 포함하지 않음
  • 전환 메커니즘이 비교적 단순; 더 정교한 중재(예: 학습된 전환, 연속 혼합)는 탐구되지 않음
  • 정책에서 명시적 힘이나 임피던스 출력 없음; 유연성은 학습된 제어가 아닌 핸드의 기계적 설계에서 기인
  • 단일 핸드 플랫폼 (ADAPT 13 DoF); 교차 구현체 평가 없음
  • 이 섹션의 다른 연구 대비 평가 범위가 제한적

결과:

집기-놓기 성공 (물체-배치 조합당 5회 시행):

방법 평균 성공 점수
VLA 전용 (고추) 0.43
VLA 전용 (테이프) 0.23
VLA + 확산 전환 0.82

VLA+확산 전환이 고추에서 VLA 전용 대비 ~90%, 테이프에서 ~257% 개선.

유연 vs. 강체 구성:

조건 유연 강체
크기 변동 있는 손잡이 회전 강체 대비 +58% 평균 개선 기준선
제어 노이즈 강건성 (가우시안 N(0,5)mm) <8% 성능 감소 74% 성능 감소
스프링 접촉 안정성 +38% 개선 기준선

다단계 정교한 작업 (각 10회 시행):

작업 성공률
물병 열고 붓기 >0.6
포스트잇 벗기고 붙이기 >0.6
다물체 파지 >0.6

초기 구성 민감도: 핸드가 목표 ~15 cm 이내에서 시작하면 >50% 성공; 15 cm 오프셋 이상에서 50% 이하로 감소 (거리당 8회 시행).

데이터 수집: 물체-배치당 20개 시연 궤적(집기-놓기); 정교한 작업당 30개 시연; ~5 Hz 샘플링 주파수.

  • 13-DoF ADAPT Hand에서 VLA + 확산 정책 전환 시연
  • 직렬 탄성 구동이 접촉 중 유용한 수동 유연성을 제공하며, 정량화된 강건성 이득
  • 공개 코드나 가중치 없음

추론 / 배포

  • 추론 지연 시간: 보고되지 않음. 시연 데이터는 ~5 Hz 샘플링 주파수로 수집되었으나, 이는 배포 제어 주파수가 아닌 데이터 캡처 속도.
  • 배포 하드웨어: UR5 로봇 팔 + 직렬 탄성 구동의 ADAPT Hand (13 DoF). Raspberry Pi 4B 유닛에서 비전/센서 처리. 추론 GPU 미명시.
  • 실시간 가능 여부: 미검증. 논문에서 자율 실행 제어 주파수나 추론 지연을 보고하지 않음. VLA와 확산 정책 간 전환 메커니즘은 활성 모드에 따라 가변 지연을 도입할 수 있음.

데이터셋 / 데이터 수집

  • 사용 데이터셋: 맞춤 시연 데이터셋. 요약에 "최소한의 시연 데이터" 언급.
  • 수집 방법: UR5 + ADAPT Hand (13 DoF) 시스템에서의 원격 조작 시연. ~5 Hz 샘플링 주파수로 수집.
  • 데이터 규모: 물체-배치 조합당 20개 시연 궤적(집기-놓기 작업); 정교한 작업(작업 A-C)당 30개 시연 궤적.
  • 원격 조작 장비: 상세히 기술되지 않음. ADAPT Hand는 직렬 탄성, 건 구동 구동을 사용; 원격 조작 인터페이스 세부 사항 미공개.
  • 데이터 포맷: 보고되지 않음.
  • 공개 여부: 아니오. 코드, 데이터 또는 모델 가중치 공개 없음.
